核心概念

多元线性回归模型

基本模型
\[Y_i = \beta_0 + \beta_1 X_{1i} + \beta_2 X_{2i} + ... + \beta_k X_{ki} + \epsilon_i\]
矩阵形式
\[\mathbf{Y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\epsilon}\]

其中：

•\(\mathbf{Y}\)：\(n \times 1\) 因变量向量
•\(\mathbf{X}\)：\(n \times (k+1)\) 设计矩阵（含常数项）
•\(\boldsymbol{\beta}\)：\((k+1) \times 1\) 参数向量
•\(\boldsymbol{\epsilon}\)：\(n \times 1\) 误差项向量
参数解释
斜率系数\(\beta_j\)
解释: 控制其他变量不变时，\(X_j\)变化1单位对\(Y\)的边际效应
截距\(\beta_0\)
解释: 所有\(X\)为零时\(Y\)的期望值
偏效应
解释: Ceteris paribus条件下的效应

OLS估计

最小二乘法原理

最小化残差平方和：

\[\min_{\boldsymbol{\beta}} \sum_{i=1}^{n}(Y_i - \mathbf{X}_i'\boldsymbol{\beta})^2 = \min_{\boldsymbol{\beta}} (\mathbf{Y} - \mathbf{X}\boldsymbol{\beta})'(\mathbf{Y} - \mathbf{X}\boldsymbol{\beta})\]
OLS估计量
\[\hat{\boldsymbol{\beta}} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{Y}\]
拟合值与残差
\[\hat{\mathbf{Y}} = \mathbf{X}\hat{\boldsymbol{\beta}} = \mathbf{P}\mathbf{Y}\]
\[\hat{\boldsymbol{\epsilon}} = \mathbf{Y} - \hat{\mathbf{Y}} = \mathbf{M}\mathbf{Y}\]

其中\(\mathbf{P} = \mathbf{X}(\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\)为投影矩阵，\(\mathbf{M} = \mathbf{I} - \mathbf{P}\)为消去矩阵。

Gauss-Markov定理与经典假设

经典假设（CLM）
假设表述数学表达违反后果
MLR.1线性于参数\(Y = X\beta + \epsilon\)模型设定错误
MLR.2随机抽样\((X_i, Y_i)\) i.i.d.标准误失效
MLR.3无完全共线性\(rank(\mathbf{X}) = k+1\)无法估计
MLR.4零条件均值\(E[\epsilonX] = 0\)
MLR.5同方差\(Var(\epsilonX) = \sigma^2\)
MLR.6正态性\(\epsilonX \sim N(0, \sigma^2)\)
Gauss-Markov定理

在MLR.1-MLR.5下，OLS估计量是最佳线性无偏估计量（BLUE）：

•Best（最佳）：方差最小
•Linear（线性）：\(\hat{\beta}\)是\(Y\)的线性函数
•Unbiased（无偏）：\(E[\hat{\beta}] = \beta\)
•Estimator（估计量）

假设检验

单个系数检验（t检验）
\[t = \frac{\hat{\beta}_j - \beta_{j,0}}{se(\hat{\beta}_j)} \sim t_{n-k-1}\]
标准误的计算
\[se(\hat{\beta}_j) = \sqrt{\hat{\sigma}^2 [(\mathbf{X}'\mathbf{X})^{-1}]_{jj}}\]

其中\(\hat{\sigma}^2 = \frac{\sum \hat{\epsilon}_i^2}{n-k-1}\)

联合假设检验（F检验）

检验\(H_0: \beta_1 = \beta_2 = ... = \beta_q = 0\)

\[F = \frac{(SSR_r - SSR_{ur})/q}{SSR_{ur}/(n-k-1)} \sim F_{q, n-k-1}\]

等价形式：

\[F = \frac{(R^2_{ur} - R^2_r)/q}{(1 - R^2_{ur})/(n-k-1)}\]
Wald检验
\[W = (\mathbf{R}\hat{\boldsymbol{\beta}} - \mathbf{r})' [\mathbf{R}(\mathbf{X}'\mathbf{X})^{-1}\mathbf{R}'\hat{\sigma}^2]^{-1} (\mathbf{R}\hat{\boldsymbol{\beta}} - \mathbf{r})\]

拟合优度

决定系数（R²）
\[R^2 = 1 - \frac{SSR}{SST} = \frac{SSE}{SST}\]

其中：

•\(SST = \sum(Y_i - \bar{Y})^2\)：总平方和
•\(SSE = \sum(\hat{Y}_i - \bar{Y})^2\)：解释平方和
•\(SSR = \sum \hat{\epsilon}_i^2\)：残差平方和
调整R²
\[\bar{R}^2 = 1 - \frac{SSR/(n-k-1)}{SST/(n-1)} = 1 - \frac{n-1}{n-k-1}(1-R^2)\]

惩罚模型复杂度，适用于模型比较。

R²的局限性
•添加变量必然增加\(R^2\)
•不同样本不可比
•不反映经济显著性
•时间序列\(R^2\)可能很高但无预测力

案例一：Fama-MacBeth两步回归

背景

检验CAPM是否能解释股票截面收益差异，估计市场风险溢价。

方法论

第一步：时间序列回归估计\(\beta\)

\[R_{it} - R_{ft} = \alpha_i + \beta_i(R_{mt} - R_{ft}) + \epsilon_{it}\]

对每只股票估计其市场\(\beta\)。

第二步：横截面回归估计风险价格

\[R_{it} - R_{ft} = \gamma_{0t} + \gamma_{1t}\hat{\beta}_i + u_{it}\]

对每个时期\(t\)进行截面回归。

第三步：时序平均

\[\bar{\gamma}_1 = \frac{1}{T}\sum_{t=1}^{T}\gamma_{1t}\]
\[se(\bar{\gamma}_1) = \frac{s_{\gamma_1}}{\sqrt{T}}\]
实证结果（美国市场1963-2023）
\(\gamma_0\)（截距）
估计值: 0.53%
t统计量: 2.34**
\(\gamma_1\)（市场风险价格）
估计值: 0.21%
t统计量: 0.89
发现
1.市场\(\beta\)的风险溢价统计不显著
2.截距显著不为零，CAPM无法完全解释收益
3.支持寻找额外定价因子

案例三：事件研究中的市场模型

背景

分析公司并购公告对股价的影响。

市场模型
\[R_{it} = \alpha_i + \beta_i R_{mt} + \epsilon_{it}\]

\[AR_{it} = R_{it} - (\hat{\alpha}_i + \hat{\beta}_i R_{mt})\]

\[t = \frac{\overline{CAR}}{\sigma_{CAR}/\sqrt{N}}\]

关键问题解答

概念类问题

Q1: 什么是Gauss-Markov定理？它的前提条件是什么？

A: Gauss-Markov定理表明，在经典线性回归假设下，OLS估计量是最佳线性无偏估计量（BLUE）。

前提条件（不需要正态假设）：

1.模型线性于参数
2.随机抽样
3.解释变量无完全共线性
4.误差项条件期望为零：\(E[\epsilon|X] = 0\)
5.同方差：\(Var(\epsilon|X) = \sigma^2\)
含义
•无偏：\(E[\hat{\beta}] = \beta\)
•最佳：在所有线性无偏估计量中方差最小

Q2: R²高是否意味着模型好？R²低是否意味着模型差？

A: 不一定。

R²高不意味模型好
•时间序列数据R²常常很高，但可能是伪回归
•过拟合可能导致高R²但差的预测
•添加无关变量也会提高R²
R²低不意味模型差
•因果效应可能存在但被噪声掩盖
•截面数据R²通常较低但结论可能可靠
•某些研究关注系数而非解释力
正确使用R²
•用调整R²比较不同变量数的模型
•关注系数的经济显著性和统计显著性
•结合样本外预测评估模型

Q3: 异方差如何影响OLS估计？

A:

系数估计
影响: 仍然无偏且一致
标准误
影响: 错误（通常偏小）
t/F检验
影响: 不可靠
置信区间
影响: 不准确
解决方案
1.使用异方差稳健标准误（首选）
2.加权最小二乘法（WLS）
3.对数变换因变量
应用类问题

Q4: 如何处理多重共线性？

A: 多重共线性的识别与处理：

识别方法
•VIF > 10表示严重共线性
•相关系数矩阵检查
•系数符号不稳定或不合理
处理策略
1.不处理：如果关注的系数不受影响
2.删除变量：删除高度相关的冗余变量
3.合并变量：构建综合指标
4.主成分分析：降维处理
5.增加样本量：如果可能
6.岭回归/LASSO：正则化方法
注意
•共线性不影响预测，只影响单个系数解释
•完全共线性才导致无法估计

Q5: Fama-MacBeth方法相比普通面板回归有什么优势？

A:

截面相关
Fama-MacBeth: 自然处理
普通面板回归: 需要聚类标准误
时序变化
Fama-MacBeth: 风险价格可时变
普通面板回归: 假设恒定
标准误
Fama-MacBeth: 简单计算
普通面板回归: 需要复杂调整
解释性
Fama-MacBeth: 直观（风险价格）
普通面板回归: 系数混合效应
适用场景
•资产定价检验
•截面收益预测
•因子模型评估
局限
•第一步估计的\(\beta\)有误差
•样本期较短时标准误不可靠

Q6: 如何选择稳健标准误的类型？

A: 根据数据特征选择：

独立截面
推荐标准误: HC1/HC3
原因: 异方差稳健
面板（个体效应）
推荐标准误: 按个体聚类
原因: 组内相关
面板（时间效应）
推荐标准误: 按时间聚类
原因: 截面相关
面板（双向）
推荐标准误: 双向聚类
原因: 两种相关
时间序列
推荐标准误: HAC (Newey-West)
原因: 自相关
实践建议
•金融面板数据：按公司和时间双向聚类
•至少40-50个聚类才可靠
•聚类数少时使用wild bootstrap
批判性问题

Q7: 统计显著与经济显著有何区别？为什么这个区分重要？

A:

统计显著性
•拒绝系数为零的原假设
•取决于样本量：大样本几乎任何非零效应都显著
•用p值或t统计量衡量
经济显著性
•效应大小是否有实际意义
•与样本量无关
•需要基于领域知识判断
例子

研究发现独立董事比例每增加1%，Tobin's Q增加0.001（t=3.5***）。

统计显著：是（p<0.01）

经济显著：否（效应太小，无实际意义）

重要性
•大数据时代统计显著很容易达到
•政策建议需要基于经济显著性
•期刊要求报告效应大小

Q8: 为什么不能随意添加控制变量？

A: 控制变量选择需要谨慎：

过度控制（Bad Control）问题
•控制了因果路径上的中介变量
•会低估真实效应
•例：研究教育对收入的影响时控制职业
遗漏变量偏差
•遗漏了混淆因素
•会高估或低估效应
•例：研究吸烟对健康的影响时未控制收入
正确做法
1.基于因果图（DAG）选择控制变量
2.只控制混淆因素（共同原因）
3.不控制中介变量或结果变量
4.考虑控制变量本身的内生性

Q9: OLS估计在什么情况下会失效？有哪些替代方法？

A:

OLS失效情况及替代
内生性
OLS结果: 有偏、不一致
替代方法: IV、DID、RDD
非线性关系
OLS结果: 错误的边际效应
替代方法: 非线性模型
二元因变量
OLS结果: 预测值超出[0,1]
替代方法: Logit/Probit
删失数据
OLS结果: 有偏
替代方法: Tobit模型
样本选择
OLS结果: 有偏
替代方法: Heckman两步法
因变量非负
OLS结果: 可能预测负值
替代方法: Poisson回归
内生性是最严重的问题
•遗漏变量
•测量误差
•同时性
•需要工具变量或自然实验
