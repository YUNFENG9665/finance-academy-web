核心概念

1

资本预算的重要性

资本预算（Capital Budgeting）是企业最重要的财务决策之一：

金额大
说明: 通常占企业资产的重要部分
影响: 决定企业规模
周期长
说明: 影响持续多年
影响: 决定长期竞争力
不可逆
说明: 难以轻易撤回
影响: 试错成本高
战略性
说明: 体现企业战略方向
影响: 影响市场地位
资本预算流程
1. 项目识别
内容: 发现投资机会
责任方: 业务部门
2. 现金流预测
内容: 估计收入和成本
责任方: 财务+业务
3. 项目评估
内容: NPV/IRR分析
责任方: 财务部门
4. 项目选择
内容: 资本配置决策
责任方: 管理层/董事会
5. 事后审计
内容: 实际vs预测
责任方: 财务部门
2

净现值法（NPV）

NPV的定义与计算
\[NPV = \sum_{t=0}^{n} \frac{CF_t}{(1+r)^t} = -I_0 + \sum_{t=1}^{n} \frac{CF_t}{(1+r)^t}\]
决策规则
NPV > 0
决策: 接受项目
含义: 创造股东价值
NPV < 0
决策: 拒绝项目
含义: 毁灭股东价值
NPV = 0
决策: 无差异
含义: 收益恰好覆盖成本
NPV的优点
•考虑货币时间价值
•考虑全部现金流
•直接衡量价值创造
•可加性：多项目NPV可相加
NPV的局限性
•依赖现金流预测准确性
•依赖折现率的选择
•不考虑管理灵活性
•难以处理战略价值
3

内部收益率（IRR）

IRR的定义

使NPV等于零的折现率：

\[0 = \sum_{t=0}^{n} \frac{CF_t}{(1+IRR)^t}\]
决策规则
•IRR > WACC（资本成本）：接受项目
•IRR < WACC：拒绝项目
IRR的问题与解决方案
问题描述原因解决方案
多重IRR存在多个使NPV=0的利率现金流符号多次变化使用NPV或MIRR
规模问题忽略投资规模比率指标用NPV排序
互斥项目与NPV可能矛盾再投资假设不同以NPV为准
非常规现金流可能无解或负解总现金流入<流出使用NPV
修正内部收益率（MIRR）
\[MIRR = \sqrt[n]{\frac{FV(正现金流)}{PV(负现金流)}} - 1\]

MIRR假设再投资按WACC进行，更为合理。

4

其他资本预算方法

方法定义优点缺点适用场景
回收期收回初始投资的时间简单、流动性忽略时间价值和后期现金流快速筛选
折现回收期折现后回收期考虑时间价值仍忽略后期现金流改进筛选
盈利指数(PI)NPV/投资资本约束下有用规模问题资本配给
会计收益率(ARR)平均利润/平均投资与报表一致忽略时间价值和现金流业绩考核
盈利指数（PI）
\[PI = \frac{PV(未来现金流)}{初始投资} = 1 + \frac{NPV}{I_0}\]

决策规则：PI > 1则接受。

5

增量现金流分析

增量现金流原则：只考虑因项目决策而产生的现金流变化。

\[增量CF = (收入 - 成本 - 折旧)(1-T) + 折旧 - CAPEX - \Delta NWC\]
现金流分析要点
沉没成本
是否计入: 否
原因: 已发生，与决策无关
机会成本
是否计入: 是
原因: 放弃的最佳替代方案
附带效应
是否计入: 是
原因: 对其他项目的影响
间接费用
是否计入: 看情况
原因: 仅计入增量部分
融资成本
是否计入: 否
原因: 已在WACC中考虑
通货膨胀
是否计入: 保持一致
原因: 名义vs实际
6

实物期权理论

传统NPV的局限
•假设"现在或永不"决策
•忽略管理灵活性
•低估高不确定性项目
实物期权类型
期权类型定义类比金融期权实例
延迟期权等待更多信息再投资看涨期权矿产开发权
扩张期权扩大成功项目看涨期权试点项目
收缩期权缩小失败项目看跌期权灵活产能
放弃期权终止失败项目看跌期权清算价值
转换期权改变产品或投入组合期权灵活工厂
分阶段期权分步投资复合期权研发项目
实物期权价值
\[项目总价值 = NPV_{传统} + 实物期权价值\]
\[扩展NPV = E[V^*] - I\]

其中 \(V^*\) 是考虑最优行使策略后的项目价值。

关键模型与公式

1

模型1：NPV完整计算

\[NPV = -I_0 + \sum_{t=1}^{n} \frac{CF_t}{(1+WACC)^t} + \frac{TV_n}{(1+WACC)^n}\]
终值（Terminal Value）计算

永续增长法：

\[TV = \frac{CF_{n+1}}{WACC - g} = \frac{CF_n(1+g)}{WACC - g}\]

退出倍数法：

\[TV = EBITDA_n \times 退出倍数\]
2

模型2：增量现金流模板

项目计算
销售收入+收入
减：可变成本-可变成本
减：固定成本-增量固定成本
减：折旧-折旧（非现金）
= EBIT息税前利润
减：所得税-EBIT×T
= NOPAT税后营业利润
加：折旧+折旧（加回）
减：CAPEX-资本支出
减：ΔNWC-营运资本变化
= 自由现金流FCF
3

模型3：等价年金法（EAA）

用于比较寿命不同的互斥项目：

\[EAA = \frac{NPV}{\text{年金现值因子}} = \frac{NPV \times r}{1 - (1+r)^{-n}}\]

选择EAA较高的项目。

4

模型4：实物期权定价

延迟期权的Black-Scholes近似
\[C = V \cdot N(d_1) - I \cdot e^{-rT} \cdot N(d_2)\]

其中：

•\(V\) = 项目资产现值
•\(I\) = 投资成本
•\(T\) = 期权期限
•\(\sigma\) = 项目价值波动率
\[d_1 = \frac{\ln(V/I) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}\]
\[d_2 = d_1 - \sigma\sqrt{T}\]
二叉树模型更适合实物期权
•可处理美式期权特征
•便于多阶段决策
•参数更直观

案例1：亚马逊的长期投资哲学

背景：亚马逊长期牺牲短期利润追求长期增长，挑战传统投资评估方法。

投资特点
长期视角
表现: 接受多年亏损
传统评估的挑战: 短期NPV可能为负
规模优先
表现: 大量资本支出
传统评估的挑战: 现金流压力大
平台投资
表现: AWS等基础设施
传统评估的挑战: 回报时间长
创新投入
表现: 研发支出巨大
传统评估的挑战: 高不确定性
AWS的投资决策分析
•2006年启动，初期无法预测规模
•传统NPV可能低估，因为：
•忽略扩张期权
•忽略学习曲线效应
•忽略平台生态价值
贝索斯的框架
•关注自由现金流，而非利润
•"第一类决策"（不可逆）谨慎，"第二类决策"（可逆）快速
•长期思维，接受短期批评
思考
•传统NPV能捕捉亚马逊的价值吗？
•如何评估战略性投资？
•市场为什么给亚马逊高估值？

案例2：医药公司的研发投资决策

背景：新药研发是实物期权理论的经典应用场景。

新药研发特点
阶段耗时成功率投资期权特点
临床前2-4年30%中扩张期权
临床I期1-2年70%中继续/放弃
临床II期2-3年33%高继续/放弃
临床III期2-4年25-30%很高最后决策
审批1-2年85%低上市准备
实物期权分析
\[项目价值 = \sum_{阶段} P(到达该阶段) \times [NPV(继续) \times P(成功) + NPV(放弃) \times P(失败)]\]
传统NPV vs 实物期权价值
传统NPV
项目A（低风险）: 1亿
项目B（高风险）: -0.5亿
实物期权价值
项目A（低风险）: 0.2亿
项目B（高风险）: 1亿
总价值
项目A（低风险）: 1.2亿
项目B（高风险）: 0.5亿
启示
1.高不确定性项目可能被传统NPV低估
2.分阶段投资创造期权价值
3.放弃期权可能比想象的更有价值

案例3：特斯拉Gigafactory的投资决策

背景：特斯拉的超级工厂是资本密集型投资的典型案例。

投资规模
•总投资：50-70亿美元
•产能：年产50GWh电池
•就业：超过10,000人
传统NPV分析的挑战
需求不确定
原因: 电动车市场快速变化
影响: 现金流预测困难
技术迭代
原因: 电池技术快速发展
影响: 设备可能过时
竞争格局
原因: 传统车企和新势力
影响: 市场份额不确定
政策风险
原因: 补贴政策变化
影响: 收入波动
实物期权视角
•扩张期权：成功后可复制到其他地区
•收缩期权：可调整产能利用率
•转换期权：可为其他车企代工
•学习效应：成本曲线下降
决策结果
•已建成多个Gigafactory
•电池成本大幅下降
•产能利用率持续提升
•战略价值超过财务价值

核心问题

概念理解类
1.为什么NPV被认为是最好的资本预算方法？相比其他方法有什么优势？
•直接衡量价值创造
•考虑货币时间价值
•考虑全部现金流
•可加性便于组合决策
2.IRR在什么情况下可能给出错误的决策？如何修正？
•非常规现金流导致多重IRR
•互斥项目规模不同
•再投资假设不合理
•使用NPV或MIRR
3.什么是实物期权？为什么传统NPV可能低估项目价值？
•管理灵活性的价值
•等待信息的权利
•调整决策的能力
•不确定性创造期权价值
分析应用类
4.如何预测投资项目的现金流？有哪些常见错误？
•分解为收入和成本组成部分
•常见错误：混淆会计利润与现金流
•常见错误：遗漏营运资本
•常见错误：包含沉没成本
5.不确定性很高的项目应该如何评估？敏感性分析和情景分析如何使用？
•敏感性分析：单变量变化
•情景分析：多变量组合
•蒙特卡洛模拟：概率分布
•实物期权：价值灵活性
6.如何判断一个项目是否具有显著的实物期权价值？
•高不确定性
•可分阶段投资
•存在学习机会
•有可量化的灵活性
批判思维类
7.为什么很多企业仍然使用回收期法而不是NPV？
•简单易懂
•流动性考虑
•不确定性厌恶
•激励机制扭曲
8.实物期权理论很美，但实务中如何应用？有什么困难？
•参数估计困难（波动率）
•模型复杂度高
•可能过度合理化坏项目
•缺乏市场验证
9.人工智能如何改变资本预算决策？
•更精准的现金流预测
•实时风险监控
•自动化敏感性分析
•决策支持系统
