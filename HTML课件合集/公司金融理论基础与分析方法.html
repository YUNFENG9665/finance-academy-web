核心概念

1

公司金融的核心问题

公司金融（Corporate Finance）回答企业的三大核心财务决策：

决策类型核心问题关键指标对应章节
投资决策投资什么项目？NPV、IRR、ROIC第4章
融资决策如何筹集资金？资本结构、WACC第3、5章
股利决策如何分配利润？股利政策、回购第6章

目标函数：股东价值最大化（Shareholder Value Maximization）

\[企业价值 = \sum_{t=1}^{\infty} \frac{E(FCF_t)}{(1+WACC)^t}\]

为什么是股东价值而非利润最大化？

•利润是会计概念，容易操纵
•利润不考虑货币时间价值
•利润不考虑风险调整
•股东是剩余索取权人，保护股东利益激励效率

财务报表分析体系

三大财务报表的关系
报表反映内容时点/期间核心等式
资产负债表财务状况时点资产 = 负债 + 权益
利润表经营成果期间收入 - 费用 = 利润
现金流量表现金变动期间期初现金 + 现金流 = 期末现金
财务报表的勾稽关系
\[\Delta 留存收益 = 净利润 - 股利\]
\[\Delta 现金 = 经营现金流 + 投资现金流 + 融资现金流\]
关键财务恒等式
\[资产 = 负债 + 所有者权益\]
\[总资产 = 流动资产 + 固定资产 + 无形资产\]

杜邦分析体系

经典杜邦分解（三因素）
\[ROE = \frac{净利润}{销售收入} \times \frac{销售收入}{总资产} \times \frac{总资产}{股东权益}\]
\[ROE = 净利率 \times 资产周转率 \times 权益乘数\]
扩展杜邦分解（五因素）
\[ROE = \frac{净利润}{EBT} \times \frac{EBT}{EBIT} \times \frac{EBIT}{销售收入} \times \frac{销售收入}{总资产} \times \frac{总资产}{股东权益}\]
税负保留率
含义: 净利润/EBT
影响因素: 税率政策
利息保留率
含义: EBT/EBIT
影响因素: 资本结构
营业利润率
含义: EBIT/销售收入
影响因素: 成本控制
资产周转率
含义: 销售收入/总资产
影响因素: 运营效率
权益乘数
含义: 总资产/股东权益
影响因素: 财务杠杆
5

自由现金流分析

企业自由现金流（FCFF）
\[FCFF = EBIT(1-T) + D\&A - CAPEX - \Delta NWC\]
股权自由现金流（FCFE）
\[FCFE = FCFF - 利息(1-T) + 净借款\]
另一种计算方式
\[FCFE = 净利润 + D\&A - CAPEX - \Delta NWC + 净借款\]
EBIT
定义: 息税前利润
理解要点: 不受资本结构影响的盈利
D&A
定义: 折旧与摊销
理解要点: 非现金费用，需加回
CAPEX
定义: 资本支出
理解要点: 维持和增长所需投资
ΔNWC
定义: 营运资本变化
理解要点: 流动资产-流动负债的变化
现金流 vs 利润的重要区别
•利润可以通过会计政策操纵
•现金流更难造假（"Cash is King"）
•利润不等于可分配的真实盈余
•应计项目可能与现金收付脱节

关键模型与公式

1

模型1：资产负债表结构模型

\[Assets = Liabilities + Equity\]
资产侧分解
\[总资产 = 现金 + 应收账款 + 存货 + 固定资产 + 其他资产\]
负债侧分解
\[负债 = 流动负债 + 长期负债\]
\[权益 = 股本 + 资本公积 + 留存收益\]
2

模型2：净营运资本分析

\[NWC = 流动资产 - 流动负债\]
\[NWC = (现金 + 应收账款 + 存货) - (应付账款 + 预收款项 + 短期借款)\]
净营运资本的意义
•NWC > 0：短期资产覆盖短期负债，流动性好
•NWC < 0：可能面临短期偿债压力
•过高的NWC可能意味着资金使用效率低
3

模型3：EBITDA与企业价值

\[EBITDA = 营业利润 + 折旧 + 摊销\]
\[EBITDA = 收入 - 经营成本（不含D\&A）\]
企业价值（EV）
\[EV = 市值 + 净负债\]
\[EV = 市值 + 有息负债 - 现金\]
EV/EBITDA倍数的意义
•去除资本结构和折旧政策影响
•便于跨公司、跨国比较
•反映收购者需支付的倍数
4

模型4：可持续增长率

\[g^* = ROE \times (1 - 股利支付率) = ROE \times b\]

其中 \(b\) 为留存比例（Plowback Ratio）。

可持续增长的含义
•不改变资本结构时企业能维持的最大增长率
•超过此增长率需要外部融资
•增长过快可能带来财务压力

案例1：苹果公司的财务策略演变

背景：苹果是全球市值最高的公司之一，其财务策略经历了显著演变。

财务特点分析
指标2015年2023年变化趋势
毛利率40.1%44.1%上升
现金储备2060亿1620亿下降
负债率35%52%上升
回购规模350亿900亿/年大幅上升
策略转变
1.从零负债转向适度举债，利用税盾
2.从囤积现金转向积极回购股票
3.开始派发股息，平衡成长与回报
分析要点
•为什么苹果不把现金用于大规模收购？
•高毛利率反映什么竞争优势？
•回购与股息如何创造股东价值？

案例2：瑞幸咖啡财务造假事件

背景：2020年4月，瑞幸自曝财务造假22亿元人民币。

造假手法
虚增交易
具体操作: 关联方虚假采购
识别信号: 现金流与收入不匹配
虚增收入
具体操作: 虚假销售订单
识别信号: 单店收入异常增长
虚减成本
具体操作: 补贴计入其他科目
识别信号: 毛利率异常高
识别财务造假的信号
1.经营现金流与净利润长期背离
2.应收账款增速远超收入增速
3.关联交易占比异常高
4.审计师变更或保留意见
5.管理层频繁减持
教训
•现金流验证收入的重要性
•实地调研的价值（浑水报告）
•财务比率交叉验证

案例3：恒大集团的高杠杆困境

背景：恒大曾是中国最大的房企，2021年陷入债务危机。

财务状况演变
指标2018年2021年风险信号
负债率83%89%持续攀升
净负债率152%199%远超红线
现金短债比0.9x0.6x流动性紧张
有息负债6730亿8700亿规模巨大
三道红线政策
\[剔除预收款后资产负债率 < 70\%\]
\[净负债率 < 100\%\]
\[现金短债比 > 1\]
分析要点
1.高杠杆模式的脆弱性
2.预售制下现金流错配风险
3.财务杠杆与经营风险叠加
4.宏观调控对企业的影响

核心问题

概念理解类
1.为什么公司金融的目标是股东价值最大化，而不是利润最大化？
•考虑货币时间价值
•考虑风险调整
•避免会计操纵
•股东作为剩余索取权人
2.净利润和自由现金流有什么区别？为什么分析师更关注现金流？
•利润包含非现金项目
•利润可被会计政策操纵
•现金流反映真实可分配价值
3.ROE高一定意味着公司优秀吗？高杠杆如何影响ROE？
•杜邦分解揭示ROE来源
•高杠杆可以"粉饰"ROE
•需结合风险评估
分析应用类
4.如何通过财务报表识别公司的财务风险信号？
•流动性指标恶化
•现金流与利润背离
•关联交易异常
5.杜邦分析如何帮助理解不同行业的商业模式差异？
•零售业：高周转、低利润率
•奢侈品：低周转、高利润率
•银行业：高杠杆
6.如何评估一家公司的自由现金流质量？
•现金流稳定性
•资本支出的维持vs增长部分
•营运资本变化的持续性
批判思维类
7.财务报表能完整反映公司价值吗？有哪些重要信息未被包含？
•无形资产低估
•人力资本未计量
•未来增长期权
8.ESG因素应该如何纳入财务分析框架？
•环境风险定价
•社会责任成本/收益
•治理溢价
9.人工智能如何改变传统财务分析方法？
•大数据异常检测
•自动化报告生成
•预测模型优化
