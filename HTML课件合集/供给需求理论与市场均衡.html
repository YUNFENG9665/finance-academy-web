核心概念

需求曲线的微观基础

需求曲线不是凭空画出来的线条，它背后是千百万消费者的理性选择。理解需求曲线，就是理解"人为什么这样买东西"。

需求的本质是什么？

需求不是"想要"，而是"愿意且能够购买"。一个穷学生想要法拉利，但这不构成需求；他愿意花30元买一杯奶茶，这才是需求。

经济学中的需求必须满足两个条件：

1.支付意愿（Willingness to Pay）：消费者主观上愿意购买
2.支付能力（Ability to Pay）：消费者客观上有足够的购买力
需求函数的完整表达

需求不仅仅取决于价格，完整的需求函数应该写作：

\[Q_d = f(P, I, P_s, P_c, T, E, N)\]

其中：

变量含义影响方向
P本品价格负向（沿曲线移动）
I消费者收入正常品正向，低档品负向
\(P_s\)替代品价格正向
\(P_c\)互补品价格负向
T消费者偏好/品味取决于偏好方向
E预期（未来价格、收入）取决于预期内容
N市场中消费者数量正向

需求曲线为什么向下倾斜？

三个层次的解释：

边际效用递减
核心逻辑: 多吃一个苹果的满足感不如第一个
深层含义: 消费者愿意为额外单位支付的价格递减
替代效应
核心逻辑: 价格上涨时转向替代品
深层含义: 价格是相对的，不是绝对的
收入效应
核心逻辑: 价格上涨降低实际购买力
深层含义: 名义收入不变但"变穷了"
替代效应与收入效应的深入分析

当商品价格变化时，消费者的需求量变化可以分解为两部分：

\[\Delta Q = \Delta Q_{替代效应} + \Delta Q_{收入效应}\]
斯勒茨基方程（Slutsky Equation）
\[\frac{\partial x}{\partial p} = \frac{\partial x^h}{\partial p} - x \cdot \frac{\partial x}{\partial m}\]

其中：

•左边是总效应（需求对价格的反应）
•第一项是替代效应（希克斯需求）
•第二项是收入效应
不同商品的效应组合
商品类型替代效应收入效应总效应需求曲线
正常品负负负向下倾斜
低档品负正通常为负向下倾斜
吉芬品负正（强）正向上倾斜

吉芬品是理论上存在但现实中极为罕见的特例——价格上涨反而导致需求量增加。经典案例是19世纪爱尔兰大饥荒时期的土豆。

需求曲线移动 vs 沿曲线移动
•价格变化 → 沿曲线移动（需求量变化）
•其他因素变化（收入、偏好、相关品价格、预期）→ 整条曲线移动（需求变化）

这个区分极其重要！新闻说"需求下降"，到底是价格上涨导致的需求量下降，还是整条需求曲线左移？这是完全不同的两件事。

需求的价格弹性公式推导

需求的价格弹性定义为需求量变化的百分比除以价格变化的百分比：

\[E_d = \frac{\Delta Q / Q}{\Delta P / P} = \frac{\Delta Q}{\Delta P} \cdot \frac{P}{Q}\]

对于连续函数，使用微分形式：

\[E_d = \frac{dQ}{dP} \cdot \frac{P}{Q}\]

对于线性需求函数 \(Q = a - bP\)：

\[E_d = -b \cdot \frac{P}{a - bP}\]

注意弹性沿着线性需求曲线是变化的！

•在高价格处（需求曲线上部）：\(|E_d| > 1\)，需求富有弹性
•在中点处：\(|E_d| = 1\)，单位弹性
•在低价格处（需求曲线下部）：\(|E_d| < 1\)，需求缺乏弹性

供给曲线的微观基础

供给曲线反映的是生产者的利润最大化决策：在什么价格下，愿意生产多少？

供给函数的完整表达
\[Q_s = g(P, P_i, T, E, N_f, G)\]

其中：

P
含义: 产品价格
影响方向: 正向（沿曲线移动）
\(P_i\)
含义: 投入品价格
影响方向: 负向
T
含义: 生产技术水平
影响方向: 正向
E
含义: 生产者预期
影响方向: 取决于预期内容
\(N_f\)
含义: 市场中企业数量
影响方向: 正向
G
含义: 政府政策（税收、补贴）
影响方向: 取决于政策类型

供给曲线为什么向上倾斜？

核心原因是边际成本递增：

•产能有限：工厂满负荷运转效率下降
•要素稀缺：扩大生产需要高价雇用额外工人
•技术约束：产量越高，单位成本越高
•机会成本：资源用于生产此商品就不能生产其他商品
利润最大化条件的推导

企业利润函数：

\[\pi = TR - TC = P \cdot Q - C(Q)\]

利润最大化的一阶条件（对Q求导并令其为零）：

\[\frac{d\pi}{dQ} = P - MC = 0\]
\[P = MC\]

这就是供给曲线的微观基础：在完全竞争市场中，企业的供给曲线就是其边际成本曲线（在平均可变成本以上的部分）。

短期供给 vs 长期供给
时间维度可调整因素供给弹性典型案例
瞬时（Market Period）几乎无法调整完全无弹性渔船已经出海，当天的鱼获量固定
短期（Short Run）可变投入（劳动、原材料）较低弹性工厂加班生产，但不能扩建
长期（Long Run）所有投入（包括资本、技术）较高弹性新工厂建设、新企业进入
超长期（Very Long Run）技术水平可变最高弹性技术革新改变生产函数

这解释了为什么石油价格暴涨时，短期供给几乎无法增加，但长期会有新油田开发、替代能源出现。

供给弹性的计算

供给的价格弹性：

\[E_s = \frac{\Delta Q_s / Q_s}{\Delta P / P} = \frac{\Delta Q_s}{\Delta P} \cdot \frac{P}{Q_s}\]

对于线性供给函数 \(Q_s = c + dP\)：

\[E_s = d \cdot \frac{P}{c + dP}\]
供给弹性的决定因素
时间长度
弹性影响: 时间越长，弹性越大
解释: 长期可以调整更多因素
生产要素可得性
弹性影响: 要素越容易获得，弹性越大
解释: 扩产更容易
库存水平
弹性影响: 库存越高，短期弹性越大
解释: 可以快速出货
生产周期
弹性影响: 周期越短，弹性越大
解释: 可以快速调整产量

市场均衡：供需交汇的奇迹

市场均衡是经济学最优美的概念之一：没有任何人"决定"价格，但价格就是自然而然地形成了。

均衡的定义
•均衡价格（P\*）：使供给量等于需求量的价格
•均衡数量（Q\*）：在均衡价格下的交易量
•市场出清：没有过剩库存，没有排队抢购
均衡的数学求解

设需求函数为 \(Q_d = D(P)\)，供给函数为 \(Q_s = S(P)\)

均衡条件：\(D(P^*) = S(P^*)\)

对于线性函数：

\[Q_d = a - bP, \quad Q_s = c + dP\]
\[a - bP^* = c + dP^*\]
\[P^* = \frac{a - c}{b + d}\]
\[Q^* = \frac{ad + bc}{b + d}\]
均衡的稳定性——瓦尔拉斯调整过程
价格高于均衡 → 供给 > 需求 → 库存积压 → 卖家降价 → 价格回落
价格低于均衡 → 需求 > 供给 → 排队抢购 → 卖家涨价 → 价格上升
马歇尔调整过程

除了瓦尔拉斯的价格调整，马歇尔提出了数量调整过程：

数量低于均衡 → 需求价格 > 供给价格 → 有利可图 → 厂商增产 → 数量增加
数量高于均衡 → 需求价格 < 供给价格 → 亏损 → 厂商减产 → 数量减少
均衡稳定性的数学条件

瓦尔拉斯稳定性要求：在均衡点附近，过度需求函数（\(ED = Q_d - Q_s\)）关于价格的斜率为负：

\[\frac{dED}{dP} = \frac{dQ_d}{dP} - \frac{dQ_s}{dP} < 0\]

即需求曲线比供给曲线更陡峭（在绝对值意义上）。

均衡的哲学意义

亚当·斯密的"看不见的手"在这里得到最清晰的体现——每个人都在追求自己的利益，但市场价格机制却让社会资源得到有效配置。

均衡的唯一性与多重均衡

在标准情况下（需求向下、供给向上），均衡是唯一的。但在特殊情况下可能存在多重均衡：

•后弯的劳动供给曲线
•网络效应导致的多重需求均衡
•递增规模报酬导致的向下倾斜供给曲线

比较静态分析

比较静态分析是经济学家的核心工具：当外部条件改变时，均衡如何变化？

比较静态分析的方法论

比较静态分析比较的是两个均衡状态，而不是从一个均衡到另一个均衡的动态过程。

基本步骤：

1.确定初始均衡（\(P_0^*, Q_0^*\)）
2.识别外生变量的变化
3.判断哪条曲线移动，向哪个方向移动
4.找到新均衡（\(P_1^*, Q_1^*\)）
5.比较新旧均衡的差异
四种基本情形
变化曲线移动价格变化数量变化
需求增加（收入↑、偏好↑）需求曲线右移P↑Q↑
需求减少（替代品降价）需求曲线左移P↓Q↓
供给增加（技术进步、成本↓）供给曲线右移P↓Q↑
供给减少（原材料涨价）供给曲线左移P↑Q↓
变化幅度的数学分析

对于线性模型，设需求函数参数 \(a\) 增加 \(\Delta a\)（需求曲线右移）：

新均衡价格：\(P_1^* = \frac{(a + \Delta a) - c}{b + d} = P_0^* + \frac{\Delta a}{b + d}\)

价格变化：\(\Delta P = \frac{\Delta a}{b + d}\)

数量变化：\(\Delta Q = d \cdot \Delta P = \frac{d \cdot \Delta a}{b + d}\)

变化幅度与弹性的关系

需求变化导致的价格变化幅度取决于供给弹性：

•供给越缺乏弹性（\(d\)越小），价格变化越大
•供给越富有弹性（\(d\)越大），数量变化越大
同时变动的复杂情形

当供需同时变化时，价格和数量的变化方向取决于哪条曲线移动更多：

需求增加 + 供给增加
价格变化: 不确定
数量变化: Q一定↑
需求减少 + 供给减少
价格变化: 不确定
数量变化: Q一定↓
需求增加 + 供给减少
价格变化: P一定↑
数量变化: 不确定
需求减少 + 供给增加
价格变化: P一定↓
数量变化: 不确定
比较静态分析的矩阵形式

对于一般形式的供需系统，可以使用全微分和克莱姆法则求解：

设 \(Q_d = D(P, \alpha)\)，\(Q_s = S(P, \beta)\)，其中 \(\alpha, \beta\) 为外生参数。

均衡条件：\(D(P, \alpha) = S(P, \beta)\)

对 \(\alpha\) 全微分：

\[D_P \cdot dP + D_\alpha \cdot d\alpha = S_P \cdot dP\]
\[\frac{dP}{d\alpha} = \frac{D_\alpha}{S_P - D_P}\]

由于 \(D_P < 0\), \(S_P > 0\)，所以 \(S_P - D_P > 0\)。若 \(D_\alpha > 0\)（需求增加参数），则 \(\frac{dP}{d\alpha} > 0\)。

消费者剩余与生产者剩余

剩余分析是评价市场效率和政策效果的核心工具。

消费者剩余（CS）

消费者愿意支付的最高价格与实际支付价格之差的总和。

•几何意义：需求曲线以下、价格线以上的面积
•经济含义：消费者从交易中获得的"净收益"
消费者剩余的数学表达

对于连续需求曲线 \(P = D^{-1}(Q)\)：

\[CS = \int_0^{Q^*} D^{-1}(Q) dQ - P^* \cdot Q^*\]

对于线性需求曲线 \(P = a - bQ\)：

\[CS = \frac{1}{2} \cdot Q^* \cdot (a - P^*) = \frac{1}{2} \cdot Q^* \cdot b \cdot Q^* = \frac{b(Q^*)^2}{2}\]

或者用价格表示：

\[CS = \frac{(a - P^*)^2}{2b}\]
生产者剩余（PS）

实际收到的价格与愿意接受的最低价格之差的总和。

•几何意义：价格线以下、供给曲线以上的面积
•经济含义：生产者从交易中获得的"净收益"
生产者剩余与利润的关系

在短期内：\(PS = TR - TVC = \pi + TFC\)

即生产者剩余等于利润加上固定成本。当固定成本为零（长期均衡）时，\(PS = \pi\)。

生产者剩余的数学表达

对于连续供给曲线 \(P = S^{-1}(Q)\)：

\[PS = P^* \cdot Q^* - \int_0^{Q^*} S^{-1}(Q) dQ\]

对于线性供给曲线 \(P = c + dQ\)：

\[PS = \frac{1}{2} \cdot Q^* \cdot (P^* - c) = \frac{d(Q^*)^2}{2}\]
总剩余与效率
\[总剩余 = CS + PS\]

在完全竞争市场的均衡点，总剩余最大化——这就是市场效率的含义。

社会总剩余最大化的证明

设社会福利函数为：

\[W = \int_0^Q D^{-1}(q) dq - \int_0^Q S^{-1}(q) dq\]

最大化条件（对Q求导）：

\[\frac{dW}{dQ} = D^{-1}(Q) - S^{-1}(Q) = 0\]
\[D^{-1}(Q^*) = S^{-1}(Q^*)\]

这正是市场均衡条件！因此，竞争市场均衡实现了社会福利最大化。

无谓损失（Deadweight Loss）

当市场偏离均衡（如价格管制、税收）时，部分社会剩余永久丧失，称为无谓损失。

\[DWL = \frac{1}{2} \times \Delta P \times \Delta Q\]

其中 \(\Delta P\) 是价格扭曲程度，\(\Delta Q\) 是数量减少量。

关键模型与公式

模型1：线性供需模型求解

需求函数：\(Q_d = a - bP\)（a > 0, b > 0）

供给函数：\(Q_s = c + dP\)（c可正可负, d > 0）

参数的经济含义
a
含义: 需求曲线的横截距（最大需求量）
典型范围: 取决于市场规模
b
含义: 需求曲线斜率的绝对值
典型范围: 反映需求对价格的敏感度
c
含义: 供给曲线的横截距
典型范围: 可正可负，反映最低生产意愿
d
含义: 供给曲线斜率
典型范围: 反映供给对价格的敏感度

均衡求解：令 \(Q_d = Q_s\)

\[a - bP^* = c + dP^*\]
\[P^* = \frac{a - c}{b + d}\]
\[Q^* = a - b \cdot \frac{a-c}{b+d} = \frac{ad + bc}{b + d}\]
数值例子

若 \(Q_d = 100 - 2P\)，\(Q_s = 20 + 3P\)

则 \(P^* = \frac{100-20}{2+3} = 16\)，\(Q^* = 100 - 2(16) = 68\)

验证：\(Q_s = 20 + 3(16) = 68\)，与需求量相等，市场出清。

模型2：消费者剩余计算

对于线性需求曲线 \(P = \frac{a}{b} - \frac{1}{b}Q\)：

\[CS = \frac{1}{2} \times (P_{max} - P^*) \times Q^*\]

其中 \(P_{max} = \frac{a}{b}\) 是需求曲线的纵截距（最高支付意愿）。

续上例的计算
•\(P_{max} = \frac{100}{2} = 50\)
•\(CS = \frac{1}{2} \times (50 - 16) \times 68 = \frac{1}{2} \times 34 \times 68 = 1156\)
生产者剩余计算
•\(P_{min} = \frac{-c}{d} = \frac{-20}{3} \approx -6.67\)（供给曲线与价格轴的交点）
•由于 \(c > 0\)，供给曲线不经过原点，需要注意积分下限
•\(PS = \frac{1}{2} \times (P^* - P_{min}) \times Q^* = \frac{1}{2} \times 68 \times 68 / 3 \times 3 = \frac{68^2}{2 \times 3} \times 3\)

更简单的方法：用供给曲线从0到Q*的面积

\[PS = P^* \cdot Q^* - \int_0^{Q^*} (c + dQ)/d \cdot dQ\]

对于 \(P = \frac{Q_s - c}{d}\)，即 \(Q_s = c + dP\)：

当 \(Q = 0\) 时，\(P = -c/d\)

\[PS = \frac{(P^* + c/d) \cdot Q^*}{2} = \frac{(16 + 20/3) \cdot 68}{2} = \frac{68/3 \cdot 68}{2} = \frac{68^2}{2 \times 3} \times (3 + 20/16)\]

简化计算：\(PS = \frac{1}{2} \times Q^* \times (P^* - MC(0))\)

其中 \(MC(0) = c/d\) 为生产第一单位的边际成本。

模型3：税收效果分析

政府对每单位商品征收税额 \(t\)：

税后均衡分析

假设税收由卖方缴纳，供给曲线垂直上移 \(t\) 单位：

•原供给曲线：\(Q_s = c + dP\)
•税后供给曲线：\(Q_s = c + d(P - t) = (c - dt) + dP\)

新均衡：

\[P_c = \frac{a - (c - dt)}{b + d} = P^* + \frac{dt}{b + d}\]
\[Q_t = a - bP_c = Q^* - \frac{bdt}{b + d}\]
税负分摊
•消费者支付价格上涨：\(P_c - P^* = \frac{dt}{b + d} = \frac{t \cdot E_s}{E_s + |E_d|}\)
•生产者收到价格下降：\(P^* - P_p = \frac{bt}{b + d} = \frac{t \cdot |E_d|}{E_s + |E_d|}\)
•税收收入：\(T = t \times Q_t\)
•无谓损失：\(DWL = \frac{1}{2} \times t \times (Q^* - Q_t)\)

关键结论：税负分摊取决于弹性！弹性小的一方承担更多税负。

税负分摊的直觉解释
•弹性小意味着对价格变化不敏感
•不敏感的一方无法通过调整行为来"躲避"税负
•因此只能被动承担更多税收
数值例子

设 \(t = 6\)，使用前面的参数（\(b = 2\), \(d = 3\)）：

•消费者承担：\(\frac{6 \times 3}{2 + 3} = 3.6\)
•生产者承担：\(\frac{6 \times 2}{2 + 3} = 2.4\)
•新均衡数量：\(Q_t = 68 - \frac{2 \times 3 \times 6}{5} = 68 - 7.2 = 60.8\)
•税收收入：\(T = 6 \times 60.8 = 364.8\)
•无谓损失：\(DWL = \frac{1}{2} \times 6 \times 7.2 = 21.6\)

模型4：价格管制效果

价格上限（Price Ceiling）：\(P_{ceiling} < P^*\)

•结果：短缺（\(Q_d > Q_s\)）
•短缺量 = \(Q_d(P_{ceiling}) - Q_s(P_{ceiling})\)
价格上限的福利分析
短缺导致的非效率：
1. 消费者剩余：部分增加（低价受益），部分减少（买不到）
2. 生产者剩余：明确减少
3. 无谓损失：交易量减少导致的社会福利损失
4. 隐性成本：排队、寻找、黑市等

价格下限（Price Floor）：\(P_{floor} > P^*\)

•结果：过剩（\(Q_s > Q_d\)）
•过剩量 = \(Q_s(P_{floor}) - Q_d(P_{floor})\)
最低工资的案例分析
•价格下限 = 最低工资
•劳动需求（企业）向下倾斜
•劳动供给（工人）向上倾斜
•最低工资高于均衡工资 → 失业（劳动过剩）

模型5：补贴效果分析

政府对每单位商品给予生产者补贴 \(s\)：

补贴后均衡
•供给曲线下移 \(s\) 单位
•新均衡价格：\(P_c = P^* - \frac{bs}{b + d}\)（消费者支付价格下降）
•生产者收到：\(P_p = P_c + s = P^* + \frac{ds}{b + d}\)（高于原均衡价格）
•新均衡数量：\(Q_s = Q^* + \frac{bds}{b + d}\)（增加）

补贴的财政成本：\(G = s \times Q_s\)

补贴的效率分析
•消费者剩余增加
•生产者剩余增加
•政府支出为负（财政成本）
•净福利变化 = \(\Delta CS + \Delta PS - G < 0\)（存在无谓损失）

应用1：股票市场的供需分析

股票的需求
•来源：投资者对公司未来现金流的预期
•影响因素：盈利预期、利率水平、风险偏好、替代投资品收益
需求曲线的特殊性

股票需求曲线可能不是简单的向下倾斜。在某些情况下，价格上涨可能增加需求（动量效应、追涨心理）。

股票的供给
•短期：几乎完全无弹性（已发行股票数量固定）
•长期：可通过增发、回购调整
定增（增发）的供需分析
增发前：供给固定 S₀，需求 D，均衡价格 P₀
增发公告：供给曲线右移 S₁
结果：在原价格下出现供给过剩 → 价格下跌至新均衡 P₁

这解释了为什么定增公告通常导致股价下跌。

股票回购的效果
回购前：供给 S₀，需求 D，均衡价格 P₀
回购实施：供给曲线左移 S₁
结果：在原价格下出现需求过剩 → 价格上涨至新均衡 P₁

应用2：债券市场与利率决定

债券价格与利率的反向关系
\[P = \frac{C}{1+r} + \frac{C}{(1+r)^2} + ... + \frac{C+F}{(1+r)^n}\]

债券价格上涨 ↔ 利率下降

债券市场的供需

需求方：投资者（银行、保险、基金、个人）

•需求曲线向下倾斜：价格越低（利率越高），债券越有吸引力

供给方：借款人（政府、企业）

•供给曲线向上倾斜：价格越高（利率越低），借款成本越低，愿意发行更多债券
货币政策对债券市场的影响
央行降息 → 无风险利率下降 → 债券需求增加（相对吸引力上升）
         → 需求曲线右移 → 债券价格上涨 → 收益率下降

应用3：外汇市场的供需分析

汇率决定的供需模型

设人民币/美元汇率为 E（每美元兑换的人民币数量）

美元需求（人民币供给）：

•中国进口商需要美元支付进口
•中国投资者需要美元投资海外
•汇率越低（美元越便宜），需求越大

美元供给（人民币需求）：

•美国进口商需要人民币支付进口
•美国投资者需要人民币投资中国
•汇率越高（美元越贵），愿意供给越多美元
贸易顺差的影响
中国贸易顺差 → 出口收入（美元）> 进口支出（美元）
            → 美元供给 > 美元需求（在原汇率下）
            → 美元贬值压力 / 人民币升值压力

应用4：房地产市场的供需动态

房地产需求的特殊性
•双重属性：既是消费品（居住需求）又是投资品（增值预期）
•预期驱动：价格上涨预期可能增加需求（投机需求）
•杠杆放大：贷款政策显著影响购买力
房地产供给的特殊性
•短期完全无弹性：建造周期长（2-3年）
•土地约束：核心区位土地稀缺
•政策依赖：土地出让、规划审批影响供给
房价上涨的供需解释
需求因素：收入增长、城市化、货币宽松、投资需求
供给因素：土地供应不足、建设周期滞后
结果：需求增长快于供给增长 → 持续价格上涨

应用5：大宗商品市场

石油市场的供需分析
需求侧
•短期需求弹性极低（运输依赖石油）
•长期可以转向替代能源、提高效率
供给侧
•短期供给弹性低（产能调整需要时间）
•OPEC卡特尔行为影响供给曲线
油价暴涨的机制
需求突然增加（经济复苏）或供给突然减少（地缘冲突）
  → 在短期低弹性情况下
  → 价格需要大幅上涨才能恢复均衡

这解释了为什么石油价格波动远大于一般商品。

案例1：房地产限购政策分析

背景：中国多个城市实施限购政策，限制家庭购房数量。

供需分析
限购前均衡：高需求 + 有限供给 → 高房价
限购政策：人为限制部分需求 → 需求曲线左移
短期效果：交易量下降，价格可能下跌
长期效果：
  - 供给端：开发商减少投资 → 供给曲线左移
  - 需求端：被压抑的需求积累
  - 结果：价格可能反弹
深层思考
•限购是否解决了"住房难"问题，还是只是压制了症状？
•被限制的需求去了哪里？（代持、假离婚、商住房）
•限购如何影响住房的"投资品"属性和"消费品"属性？
政策效果的定量估计

假设限购使30%的投机需求退出市场：

•需求曲线左移约30%
•若供给弹性为0.5，需求弹性为-1.0
•价格下降幅度约为需求减少幅度的2/3

案例2：2020年口罩价格管制

背景：新冠疫情初期，多国政府实施口罩价格上限。

供需分析
正常状态：需求较低，价格稳定
疫情冲击：需求曲线大幅右移 → 均衡价格暴涨
政府干预：价格上限 < 新均衡价格 → 短缺
短缺的后果
排队等待
具体表现: 凌晨排队、浪费时间
经济学解释: 时间成本替代货币成本
非价格配给
具体表现: 先到先得、关系配给
经济学解释: 价格机制失灵后的替代机制
黑市交易
具体表现: 高价转卖、执法成本
经济学解释: 被压制的价格信号在地下恢复
质量下降
具体表现: 假冒伪劣产品出现
经济学解释: 无法通过价格竞争，转向质量"竞争"
供给不足
具体表现: 生产激励不足
经济学解释: 低价抑制生产积极性
替代方案比较
政府采购+定向发放
优点: 公平分配
缺点: 效率低，反应慢
价格放开+补贴低收入者
优点: 保持激励
缺点: 执行复杂，补贴精准度
出口限制+产能扩张
优点: 增加国内供给
缺点: 国际关系影响
分时定价
优点: 平滑需求
缺点: 执行困难

案例3：Uber的动态定价（Surge Pricing）

背景：Uber在需求高峰时段提高价格，有时高达平时的2-3倍。

经济学解释
正常时段：供需平衡，正常价格
高峰时段（下雨、演唱会结束）：需求曲线右移
动态定价：价格上涨 → 两个效果
  1. 需求侧：部分乘客选择其他出行方式
  2. 供给侧：更多司机愿意上线
结果：市场趋向新均衡，减少等待时间
消费者反感的原因
•"公平"直觉：同样的服务为什么价格不同？
•信息不对称：消费者不知道涨价是否合理
•趁人之危感：在人们"必须"用车时涨价

思考：动态定价在经济学上是效率的，但为什么消费者普遍反感？这揭示了效率与公平之间的张力。

需求侧变化

中期（2022-2023年）：
  - 新产能逐步投产
  - 部分需求回落
  - 短缺缓解，价格下降

长期影响：
  - 各国推动本土芯片制造
  - 供应链重组

企业B：减排成本50元/吨
碳价：70元/吨

结果：
- 企业A选择购买配额（比自己减排便宜）
- 企业B选择减排并出售配额（获利20元/吨）
- 社会总减排成本最小化

自我评估

1

评估题1：概念辨析

下列说法是否正确？如果错误，请指出错误所在。

1."需求增加意味着需求曲线向右移动。"
2."价格上涨导致供给增加。"
3."均衡价格是政府规定的公平价格。"
4."税收全部由消费者承担。"
5."价格上限总是能帮助穷人。"
答案
1.正确
2.错误——价格上涨导致"供给量"增加（沿曲线移动），而非"供给"增加（曲线移动）
3.错误——均衡价格是市场供需自发形成的
4.错误——税负分摊取决于供需弹性
5.错误——可能导致短缺、黑市等，穷人不一定受益
2

评估题2：图形分析

画出以下情形的供需图：

1.政府对某商品征税
2.政府实施价格下限（高于均衡价格）
3.需求增加且供给减少同时发生
3

评估题3：案例分析

选择一个你熟悉的市场（如咖啡、电影票、机票），分析：

1.影响需求的主要因素有哪些？
2.影响供给的主要因素有哪些？
3.该市场是否存在政府干预？效果如何？
4.价格是否经常波动？原因是什么？
4

评估题4：批判思考

"既然市场均衡能实现效率最大化，政府就不应该干预市场。"

请评价这一观点，讨论其正确性和局限性。
