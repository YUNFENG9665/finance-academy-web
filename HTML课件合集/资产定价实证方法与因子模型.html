核心概念

资产定价实证框架

资产定价模型的一般形式
\[E[R_i] - R_f = \sum_{k=1}^{K} \beta_{ik} \lambda_k\]

其中：

•\(E[R_i] - R_f\)：资产\(i\)的期望超额收益
•\(\beta_{ik}\)：资产\(i\)对因子\(k\)的暴露（因子载荷）
•\(\lambda_k\)：因子\(k\)的风险价格
实证检验的核心问题
定价误差
检验内容: \(\alpha_i = 0\)？
方法: 时间序列检验
风险价格
检验内容: \(\lambda_k > 0\)？
方法: 横截面检验
模型比较
检验内容: 哪个模型更好？
方法: GRS检验、截面\(R^2\)

时间序列检验

回归模型
\[R_{it} - R_{ft} = \alpha_i + \sum_{k=1}^{K} \beta_{ik} F_{kt} + \epsilon_{it}\]
对于单个资产的假设检验
\[H_0: \alpha_i = 0\]
\[t_{\alpha_i} = \frac{\hat{\alpha}_i}{se(\hat{\alpha}_i)}\]

联合检验：GRS检验（Gibbons, Ross, Shanken 1989）：

\[GRS = \frac{T-N-K}{N}\left(\frac{1}{1+\bar{F}'\hat{\Sigma}_F^{-1}\bar{F}}\right)\hat{\boldsymbol{\alpha}}'\hat{\Sigma}^{-1}\hat{\boldsymbol{\alpha}} \sim F_{N, T-N-K}\]

其中：

•\(T\)：时间期数
•\(N\)：测试资产数
•\(K\)：因子数
•\(\bar{F}\)：因子均值向量
•\(\hat{\Sigma}_F\)：因子协方差矩阵
•\(\hat{\Sigma}\)：残差协方差矩阵
•\(\hat{\boldsymbol{\alpha}}\)：alpha向量
GRS检验的解释
•拒绝\(H_0\)：因子模型不能完全解释测试资产的收益
•不拒绝\(H_0\)：因子模型定价能力较好
•注意：检验功效受测试资产选择影响

横截面检验：Fama-MacBeth方法

三步法流程

第一步：时间序列回归估计Beta

\[R_{it} - R_{ft} = \alpha_i + \boldsymbol{\beta}_i' \mathbf{F}_t + \epsilon_{it}, \quad t = 1, ..., T\]

对每个资产\(i\)估计因子载荷\(\hat{\boldsymbol{\beta}}_i\)

第二步：横截面回归估计风险价格

\[R_{it} - R_{ft} = \gamma_{0t} + \boldsymbol{\gamma}_t' \hat{\boldsymbol{\beta}}_i + u_{it}, \quad \forall t\]

对每个时期\(t\)进行横截面回归，获得\(\{\gamma_{0t}, \boldsymbol{\gamma}_t\}\)

第三步：计算时间序列均值和标准误

\[\bar{\gamma}_k = \frac{1}{T}\sum_{t=1}^{T}\gamma_{kt}\]
\[se(\bar{\gamma}_k) = \frac{s_{\gamma_k}}{\sqrt{T}}\]

其中\(s_{\gamma_k} = \sqrt{\frac{1}{T-1}\sum_{t=1}^{T}(\gamma_{kt} - \bar{\gamma}_k)^2}\)

Shanken (1992) 误差中的误差调整

由于\(\hat{\beta}\)有估计误差，Fama-MacBeth标准误需要调整：

\[Var(\hat{\gamma}) = \frac{1}{T}\left[(1 + \bar{\gamma}'\hat{\Sigma}_F^{-1}\bar{\gamma})\Sigma_u + \Sigma_\gamma\right]\]

其中\(\Sigma_u\)为第二阶段残差协方差，\(\Sigma_\gamma\)为\(\gamma_t\)的时序方差。

因子构建方法

组合排序法（Fama-French方法）：

单变量排序
1.每月末按特征排序
2.分成N组（如5组或10组）
3.计算下月各组等权/市值加权收益
4.多空因子 = 高组 - 低组

双变量独立排序（2×3或5×5）：

例：SMB因子构建（规模-BM 2×3排序）

低BM中BM高BM
小市值SLSMSH
大市值BLBMBH
\[SMB = \frac{1}{3}(SL + SM + SH) - \frac{1}{3}(BL + BM + BH)\]
\[HML = \frac{1}{2}(SH + BH) - \frac{1}{2}(SL + BL)\]

条件排序（控制另一变量）：

1.先按变量A分组
2.在每组内按变量B排序
3.合并各组形成因子

关键公式与检验方法

时间序列回归的标准误

OLS标准误（假设同方差、无自相关）：

\[se(\hat{\alpha}_i) = \sqrt{\frac{\hat{\sigma}_i^2}{T}}\]

Newey-West HAC标准误（异方差和自相关稳健）：

\[\hat{V}_{HAC} = \hat{\Gamma}_0 + \sum_{j=1}^{L}\left(1-\frac{j}{L+1}\right)(\hat{\Gamma}_j + \hat{\Gamma}_j')\]

其中\(\hat{\Gamma}_j = \frac{1}{T}\sum_{t=j+1}^{T}\hat{\epsilon}_t\hat{\epsilon}_{t-j}\mathbf{x}_t\mathbf{x}_{t-j}'\)

测试资产的选择
25个规模-BM组合
优点: 经典、可比性强
缺点: 可能对FF因子有利
行业组合
优点: 经济含义清晰
缺点: 因子暴露可能较弱
个股
优点: 信息量大
缺点: 噪音大、计算量大
异象组合
优点: 针对性强
缺点: 可能过拟合
最佳实践
•使用多种测试资产
•包括模型"难以解释"的资产
•报告各组alpha的t统计量
滚动窗口估计
滚动Beta估计

使用过去\(W\)期数据估计Beta，用于下一期预测。

\[\hat{\beta}_{i,t} = \frac{\sum_{s=t-W}^{t-1}(R_{is} - \bar{R}_i)(F_s - \bar{F})}{\sum_{s=t-W}^{t-1}(F_s - \bar{F})^2}\]
窗口长度选择
•太短：估计噪音大
•太长：参数变化被忽略
•常用：36-60个月

关键问题解答

概念类问题

Q1: 时间序列检验和横截面检验有什么区别？它们各自检验什么？

A:

数据结构
时间序列检验: 单个资产的时序数据
横截面检验: 每期多个资产的截面数据
检验内容
时间序列检验: Alpha是否为零
横截面检验: 风险价格是否显著为正
原假设
时间序列检验: \(H_0: \alpha_i = 0\)
横截面检验: \(H_0: \lambda_k = 0\)
代表方法
时间序列检验: GRS检验
横截面检验: Fama-MacBeth
关系
•时间序列alpha=0 ⟺ 横截面上只有Beta决定期望收益
•两种检验应该得到一致的结论
•实际中可能因估计误差、样本选择而有差异

Q2: 为什么Fama-MacBeth方法需要Shanken调整？

A:

问题来源

Fama-MacBeth方法的第二步使用的是估计的\(\hat{\beta}\)，而非真实的\(\beta\)。

后果
•第二步回归存在误差中的误差（Errors-in-Variables）
•Fama-MacBeth标准误低估了真实的不确定性
•t统计量被高估
Shanken调整公式
\[Var_{adjusted}(\hat{\gamma}) = Var_{FM}(\hat{\gamma}) \times (1 + \bar{\gamma}'\hat{\Sigma}_F^{-1}\bar{\gamma})\]
实践建议
•因子均值和方差较大时，调整更重要
•报告调整前后的标准误
•也可使用Bootstrap方法

Q3: 什么是GRS检验？如何解释其结果？

A:

GRS检验的含义

联合检验所有测试资产的alpha是否同时为零。

\[H_0: \alpha_1 = \alpha_2 = ... = \alpha_N = 0\]
统计量
\[GRS = \frac{T-N-K}{N} \times \frac{\hat{\boldsymbol{\alpha}}'\hat{\Sigma}^{-1}\hat{\boldsymbol{\alpha}}}{1 + \bar{F}'\hat{\Sigma}_F^{-1}\bar{F}}\]
解释
•GRS高（p值小）：拒绝\(H_0\)，因子模型不能完全解释收益
•GRS低（p值大）：不拒绝\(H_0\)，模型定价能力较好
注意事项
1.测试资产选择影响结果
2.高GRS也可能因为单个异常资产
3.应结合alpha的经济大小判断

Q8: 风险解释与行为解释的争论如何理解？

A:

风险解释（Risk-based）：

•高收益是对承担系统性风险的补偿
•例：价值股在经济衰退时表现更差
•代表：Fama-French，q因子模型

行为解释（Behavioral）：

•高收益来自投资者的错误定价
•例：投资者过度外推成长股表现
•代表：Lakonishok, Shleifer, Vishny (1994)
两种解释的检验
宏观条件
风险解释预测: 不好时代超额收益更高
行为解释预测: 无明确预测
投资者情绪
风险解释预测: 无明确预测
行为解释预测: 情绪高时效应减弱
长期表现
风险解释预测: 持续存在
行为解释预测: 套利后消失
盈利公告
风险解释预测: 不应有额外反应
行为解释预测: 应有正面意外
综合观点
•两种解释可能同时存在
•不同异象可能有不同解释
•实证上难以完全区分

价值长期有效效应较弱会计信息质量
动量强且持续很弱或反转涨跌停、散户交易
盈利显著较显著相对一致
低波动显著显著彩票偏好普遍
中国市场特殊因素
1.市场微观结构：涨跌停限制、T+1制度
2.投资者结构：散户占比高、机构化程度低
3.制度因素：壳资源价值、审批制IPO
4.信息环境：会计信息质量、信息不对称
研究启示
•不能简单照搬美国因子
•需要适应中国市场特点
•关注制度变迁的影响
