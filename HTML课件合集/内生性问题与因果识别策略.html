核心概念

内生性问题

定义

当\(E[\epsilon|X] \neq 0\)时，解释变量\(X\)与误差项\(\epsilon\)相关，称\(X\)为内生变量。

内生性的后果
•OLS估计量有偏（Biased）
•OLS估计量不一致（Inconsistent）
•无法得出因果结论
内生性的来源
来源描述数学表达金融例子
遗漏变量未观测因素同时影响X和Y\(Y = \beta X + \gamma Z + \epsilon\)，Z被遗漏CEO能力影响薪酬和业绩
测量误差变量测量不准确\(X^* = X + u\)，观测到\(X^*\)财务数据的会计操纵
同时性/双向因果X影响Y，Y也影响X\(Y = \beta X + \epsilon\)，\(X = \gamma Y + \eta\)股价与交易量
样本选择样本非随机\(Y = \beta X + \epsilon\)仅在\(S=1\)时观测只研究上市公司
遗漏变量偏差公式
\[\text{plim}(\hat{\beta}_{OLS}) = \beta + \gamma \cdot \frac{Cov(X, Z)}{Var(X)}\]

偏差方向取决于：

•遗漏变量对Y的影响方向（\(\gamma\)）
•遗漏变量与X的相关性方向

工具变量法（IV）

有效工具变量的条件
条件名称数学表达解释
IV1相关性（Relevance）\(Cov(Z, X) \neq 0\)工具与内生变量相关
IV2外生性（Exogeneity）\(Cov(Z, \epsilon) = 0\)工具与误差项不相关
IV3排除性约束Z只通过X影响Y工具不直接进入方程
两阶段最小二乘法（2SLS）
第一阶段
\[X_i = \pi_0 + \pi_1 Z_i + v_i\]

获得拟合值\(\hat{X}_i\)

第二阶段
\[Y_i = \beta_0 + \beta_1 \hat{X}_i + \epsilon_i\]
IV估计量
\[\hat{\beta}_{IV} = \frac{Cov(Z, Y)}{Cov(Z, X)} = \frac{\sum(Z_i - \bar{Z})(Y_i - \bar{Y})}{\sum(Z_i - \bar{Z})(X_i - \bar{X})}\]

矩阵形式（恰好识别时）：

\[\hat{\boldsymbol{\beta}}_{IV} = (\mathbf{Z}'\mathbf{X})^{-1}\mathbf{Z}'\mathbf{Y}\]
过度识别时的GMM估计
\[\hat{\boldsymbol{\beta}}_{GMM} = (\mathbf{X}'\mathbf{Z}\mathbf{W}\mathbf{Z}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{Z}\mathbf{W}\mathbf{Z}'\mathbf{Y}\]

IV诊断检验

弱工具变量检验

第一阶段F统计量：

\[F = \frac{(R^2_{full} - R^2_{restricted})/q}{(1-R^2_{full})/(n-k-1)}\]
F > 10
判断: 工具较强
问题: 可接受
F < 10
判断: 弱工具
问题: 偏差、检验失效
F ≈ 0
判断: 不相关
问题: 无法识别

Stock-Yogo临界值：考虑最大相对偏差的临界值

过度识别检验（Sargan/Hansen J检验）
\[J = n \cdot \hat{\boldsymbol{\epsilon}}'\mathbf{Z}(\mathbf{Z}'\mathbf{Z})^{-1}\mathbf{Z}'\hat{\boldsymbol{\epsilon}} / \hat{\sigma}^2 \sim \chi^2_{m-k}\]

其中\(m\)为工具数，\(k\)为内生变量数。

•\(H_0\)：所有工具都是外生的
•拒绝\(H_0\)：至少有一个工具无效
•注意：不拒绝不意味工具一定有效

双重差分法（DID）

基本设定
\[Y_{it} = \alpha + \beta_1 Treat_i + \beta_2 Post_t + \beta_3 (Treat_i \times Post_t) + \epsilon_{it}\]
\(\beta_1\)
含义: 处理组与对照组的基期差异
\(\beta_2\)
含义: 时间趋势（共同冲击）
\(\beta_3\)
含义: DID估计量：处理效应
DID估计量的等价表达
\[\hat{\beta}_{DID} = (\bar{Y}_{T,post} - \bar{Y}_{T,pre}) - (\bar{Y}_{C,post} - \bar{Y}_{C,pre})\]
多期DID（事件研究设计）
\[Y_{it} = \alpha_i + \gamma_t + \sum_{k \neq -1} \beta_k \cdot D_{it}^k + \epsilon_{it}\]

其中\(D_{it}^k = 1\)表示个体\(i\)在时期\(t\)处于相对事件时间\(k\)。

平行趋势假设

核心识别假设：在没有处理的情况下，处理组和对照组的结果变量会有相同的时间趋势。

检验方法：

•事前趋势检验（Pre-trend test）
•安慰剂检验（Placebo test）
•图形展示动态效应

断点回归（RDD）

基本思想

利用政策门槛或规则的不连续性，在门槛附近近似随机分配。

Sharp RDD（确定性断点）：

\[D_i = \mathbf{1}(X_i \geq c)\]

Fuzzy RDD（模糊断点）：

\[P(D_i = 1 | X_i) \text{ 在 } c \text{ 处跳跃}\]
估计方法
参数方法
\[Y_i = \alpha + \beta D_i + f(X_i - c) + D_i \cdot g(X_i - c) + \epsilon_i\]

局部线性回归（推荐）：

\[\hat{\tau}_{RDD} = \lim_{x \downarrow c} E[Y|X=x] - \lim_{x \uparrow c} E[Y|X=x]\]

使用核函数加权，带宽选择很关键。

RDD的有效性检验
密度检验
目的: 检验操纵（Manipulation）
方法: McCrary检验
协变量平衡
目的: 断点处协变量连续
方法: 以协变量为Y运行RDD
安慰剂检验
目的: 假门槛无效应
方法: 在其他点估计
带宽敏感性
目的: 结果对带宽稳健
方法: 多种带宽

匹配方法

倾向得分匹配（PSM）
倾向得分定义
\[e(X) = P(D = 1 | X) = E[D | X]\]

倾向得分定理（Rosenbaum & Rubin, 1983）：

如果\((Y_0, Y_1) \perp D | X\)，则\((Y_0, Y_1) \perp D | e(X)\)

PSM步骤
1.估计倾向得分
\[\hat{e}(X_i) = \hat{P}(D_i = 1 | X_i)\]

通常使用Logit模型

2.匹配
•最近邻匹配（1:1, 1:k）
•卡尺匹配（Caliper）
•核匹配（Kernel）
3.估计ATT
\[\hat{ATT} = \frac{1}{N_T}\sum_{i: D_i=1}[Y_i - \hat{Y}_i^{(0)}]\]
逆概率加权（IPW）
\[\hat{ATT}_{IPW} = \frac{1}{N_T}\sum_{i=1}^{n}\frac{D_i \cdot Y_i}{e(X_i)} - \frac{1}{N_T}\sum_{i=1}^{n}\frac{(1-D_i) \cdot Y_i \cdot e(X_i)}{(1-e(X_i))}\]

固定效应与面板数据

个体固定效应模型
\[Y_{it} = \alpha_i + \mathbf{X}_{it}'\boldsymbol{\beta} + \epsilon_{it}\]

控制所有不随时间变化的个体特征。

时间固定效应
\[Y_{it} = \gamma_t + \mathbf{X}_{it}'\boldsymbol{\beta} + \epsilon_{it}\]

控制所有影响所有个体的时间因素。

双向固定效应
\[Y_{it} = \alpha_i + \gamma_t + \mathbf{X}_{it}'\boldsymbol{\beta} + \epsilon_{it}\]
组内估计（Within Estimator）
\[\tilde{Y}_{it} = \tilde{\mathbf{X}}_{it}'\boldsymbol{\beta} + \tilde{\epsilon}_{it}\]

其中\(\tilde{Y}_{it} = Y_{it} - \bar{Y}_i\)

研究问题

教育年限对收入的因果效应是多少？（能力偏差问题）

内生性问题
•个人能力同时影响教育选择和收入

案例二：最低工资对就业的DID分析

研究背景

Card & Krueger (1994)分析新泽西州提高最低工资对快餐店就业的影响。

自然实验设定
•处理组：新泽西州（1992年4月提高最低工资）
•对照组：宾夕法尼亚州东部（无政策变化）
•时间：政策前后各一次调查
DID模型
\[Employment_{it} = \alpha + \beta_1 NJ_i + \beta_2 After_t + \beta_3 (NJ \times After)_{it} + \epsilon_{it}\]
结果
政策前政策后差异
新泽西(NJ)20.4421.03+0.59
宾州(PA)23.3321.17-2.16
差中差--+2.75
DID估计
系数估计值标准误含义
\(\beta_3\)2.75**(1.36)最低工资提高增加就业
稳健性检验
1.平行趋势：检验政策前两组趋势相似
2.安慰剂检验：高工资店铺无显著效应
3.替代对照组：使用NJ高工资店作为对照

•结果挑战传统经济学预测
•方法论上开创DID在劳动经济学的应用

案例三：IPO定价效率的RDD分析

研究问题

监管审核门槛是否影响IPO定价效率？

制度背景

中国证监会对IPO发行市盈率有隐性指导线（如23倍PE），超过需要额外审核。

RDD设计
•运行变量：申报市盈率与门槛的距离
•门槛：23倍PE
•处理：超过门槛需要延迟审核
模型
\[Underpricing_i = \alpha + \tau D_i + \beta_1(PE_i - 23) + \beta_2 D_i(PE_i - 23) + \epsilon_i\]

其中\(D_i = \mathbf{1}(PE_i \geq 23)\)

RDD估计结果
带宽LATE估计标准误观测数
±3 PE8.5%***(2.1%)156
±5 PE7.2%***(1.8%)234
±7 PE6.8%***(1.5%)312
有效性检验
检验统计量p值结论
McCrary密度检验0.890.37无操纵
公司规模连续性0.020.89协变量平衡
行业分布连续性χ²=3.20.36无选择
发现
1.超过门槛的IPO首日抑价率显著更高
2.监管延迟导致定价信息过时
3.效应在门槛附近局部成立

关键问题解答

概念类问题

Q1: 什么是局部平均处理效应（LATE）？它与ATE有什么区别？

A:

LATE定义
\[LATE = E[Y_1 - Y_0 | Complier]\]

即工具变量只能识别响应者（Compliers）的平均处理效应。

人群分类（根据Z对D的影响）：

类型定义Z=0时DZ=1时D
Compliers响应工具01
Always-takers总是接受处理11
Never-takers从不接受处理00
Defiers逆向响应10
LATE vs ATE
•LATE仅对Compliers有效
•如果Compliers不具代表性，LATE ≠ ATE
•政策含义：IV结果可能不适用于一般人群

Q2: 如何判断工具变量的有效性？

A: 工具变量有效性的两个条件检验：

相关性（可检验）
1.第一阶段F统计量 > 10
2.Stock-Yogo弱工具临界值
3.Anderson-Rubin检验（对弱工具稳健）
外生性（不可直接检验）
1.经济直觉和理论论证
2.过度识别检验（Sargan/Hansen J）
•注意：不拒绝仅说明工具相互一致
•所有工具可能同向违反外生性
3.安慰剂检验
4.敏感性分析（如Conley bounds）
实践建议
•外生性最终依赖经济学论证
•报告多种工具的结果
•讨论潜在的违反情况

Q3: DID的平行趋势假设为什么重要？如何检验？

A:

为什么重要

平行趋势是DID因果识别的核心。如果没有政策，处理组和对照组应该有相同的趋势。违反这一假设意味着：

•DID估计混入了趋势差异
•无法区分政策效应和自然差异
检验方法
1.事前趋势图
•画出处理前两组的结果变量
•观察是否平行
2.事件研究设计
\[Y_{it} = \alpha_i + \gamma_t + \sum_{k=-K}^{-2}\beta_k D_{it}^k + \sum_{k=0}^{L}\beta_k D_{it}^k + \epsilon_{it}\]

检验\(\beta_{-K}, ..., \beta_{-2}\)是否联合为零

3.安慰剂检验
•假设政策提前发生
•不应有显著效应
注意
•事前趋势平行不保证反事实趋势平行
•趋势平行是充分非必要条件
•考虑使用合成控制或其他方法作为补充
应用类问题

Q4: 如何在金融研究中寻找有效的工具变量？

A: 金融研究中常见的IV策略：

1. 地理/距离工具

•例：到投资银行总部的距离影响融资成本
•逻辑：距离影响信息获取，但与公司特征无关

2. 监管变化

•例：SOX法案影响审计质量
•逻辑：法规变化是外生冲击

3. 同行效应

•例：同行业公司平均值作为工具
•注意：需要排除同行直接影响

4. 天气/自然灾害

•例：降雨影响投资者情绪
•逻辑：天气对人类决策有随机影响

5. 历史变量

•例：历史路径依赖的制度特征
•逻辑：历史因素与当前未观测因素无关
寻找IV的原则
•理论先行：有清晰的经济学逻辑
•检验相关性：第一阶段要强
•论证外生性：详细讨论排除性约束
•稳健性：多种工具对比

Q5: 什么时候应该使用固定效应而不是其他因果推断方法？

A:

适用FE的情况
1.面板数据可用
2.主要内生性来源是时不变的未观测异质性
3.关注的自变量有足够的组内变异
4.不需要估计时不变变量的效应
FE的局限
1.无法处理时变的遗漏变量
2.无法估计时不变解释变量的系数
3.消除了大量变异，可能降低效率
4.与测量误差结合会放大偏差
与其他方法的比较
IV
处理的内生性: 时变内生性
FE能否替代: 不能
DID
处理的内生性: 时不变选择
FE能否替代: 类似效果
RDD
处理的内生性: 门槛处选择
FE能否替代: 不能
PSM
处理的内生性: 观测混淆
FE能否替代: 有限替代

Q6: 如何选择合适的匹配方法？

A:

方法比较
方法优点缺点适用情况
PSM（最近邻）直观、简单可能匹配质量差协变量少
PSM（核匹配）利用更多信息计算复杂中等样本

IPW保留全样本极端权重敏感大样本

选择建议
1.检查倾向得分分布重叠
2.匹配后检验协变量平衡
3.报告多种匹配方法结果
4.考虑敏感性分析

Q7: 为什么"可信革命"强调研究设计而非复杂的计量方法？

A:

可信革命的核心观点
1.因果识别来自研究设计，不是统计技术
2.外生变异的来源比估计方法更重要
3.简单方法+好设计 > 复杂方法+差设计
对研究实践的影响
控制变量越多越好
新范式: 理论指导变量选择
添加多种固定效应
新范式: 明确识别策略
报告多种模型
新范式: 报告核心设计检验
强调显著性
新范式: 强调可信度
批判与反思
•自然实验可能外部效度有限
•过度追求因果可能忽略有价值的相关研究
•不是所有重要问题都有干净的识别

Q8: IV估计的LATE解释带来什么实际问题？

A:

主要问题
1.外推困难
•LATE只对Compliers有效
•政策推广时目标人群可能不同
2.工具选择影响结论
•不同工具识别不同的Compliers
•可能得到不同的LATE
3.政策建议的局限
•LATE可能不代表政策的平均效应
•成本效益分析需要谨慎
解决思路
1.理解Compliers是谁
2.使用多个工具检验稳健性
3.报告异质性处理效应
4.结合其他方法综合判断

Q9: 因果推断方法在金融研究中面临哪些特殊挑战？

A:

金融研究的特殊性
1.市场效率与预期
•理性预期使政策效应提前反映
•难以用传统DID识别
2.一般均衡效应
•处理组行为影响对照组
•违反SUTVA假设
3.内生样本选择
•上市、退市、并购都是内生决策
•幸存者偏差普遍
4.高频数据的挑战
•传统方法假设静态效应
•金融市场反应迅速
5.外部有效性问题
•自然实验样本可能特殊
•市场制度差异大
应对策略
•明确识别假设和适用条件
•使用多种方法交叉验证
•关注经济机制而非仅仅统计显著性
•讨论外部有效性和局限

▶反向因果: Y反过来影响X

工具变量|IV·2SLS
▶好的IV: 与X相关(相关性)且与ε无关(外生性)
▶两阶段最小二乘: 先预测X̂，再用X̂回归Y
▶弱工具变量: F统计量<10→IV估计不可靠
自然实验|DID·RDD
▶DID: δ=(Ȳ_T,后-Ȳ_T,前)-(Ȳ_C,后-Ȳ_C,前)
▶RDD: 在断点附近比较处理效应
▶平行趋势假设: DID有效的关键前提
匹配方法|PSM·IPTW
▶倾向得分: P(D=1|X)的估计
▶PSM: 找到特征相似的对照观测
▶双重稳健: 结合回归+加权提高估计可靠性
