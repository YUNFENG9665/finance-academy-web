货币的三大功能
交易媒介

货币层次（中国定义）：

层次构成流动性统计口径
M0流通中现金最高央行负债
M1M0 + 企业活期存款 + 机关团体存款高狭义货币

•美国：M2不含大额定期存款
•欧元区：M3更常用
•中国：M2占GDP比例全球最高（约200%）

货币供给机制

基础货币与货币乘数
\[M = m \times B\]
•\(M\)：货币供应量（如M2）
•\(B\)：基础货币（央行负债 = 流通中现金 + 准备金）
•\(m\)：货币乘数
货币乘数推导
\[m = \frac{1 + c}{c + r + e}\]
•\(c\)：现金漏损率（公众持有现金/存款）
•\(r\)：法定准备金率
•\(e\)：超额准备金率

简化形式（假设\(c=0, e=0\)）：

\[m = \frac{1}{r}\]
影响货币乘数的因素
因素变化方向对乘数影响原因
法定准备金率↑—↓银行可贷资金减少
超额准备金率↑—↓银行惜贷
现金漏损率↑—↓存款创造减少
利率↑—↑超额准备金减少
央行调控工具
工具作用机制使用频率中国特色工具
公开市场操作买卖国债调节基础货币日常逆回购、MLF
准备金率直接影响乘数偶尔定向降准
政策利率影响资金成本定期LPR
再贷款/再贴现向银行提供流动性需要时PSL、支农支小
窗口指导行政引导常态中国特色

货币需求理论

古典货币数量论
\[MV = PY\]
•\(M\)：货币供应量
•\(V\)：货币流通速度（短期稳定）
•\(P\)：价格水平
•\(Y\)：实际产出
剑桥方程
\[M = kPY\]

\(k = 1/V\)，强调货币的资产需求。

凯恩斯流动性偏好理论
交易动机
决定因素: 收入水平
与利率关系: 不敏感
预防动机
决定因素: 不确定性
与利率关系: 不敏感
投机动机
决定因素: 利率水平
与利率关系: 负相关（关键）
货币需求函数
\[L = L_1(Y) + L_2(r)\]
\[\frac{\partial L}{\partial Y} > 0, \quad \frac{\partial L}{\partial r} < 0\]

流动性陷阱：当利率极低时，货币需求无限弹性，货币政策失效。

弗里德曼货币需求函数
\[\frac{M^d}{P} = f(Y_p, r_b - r_m, r_e - r_m, \pi^e, w)\]
•\(Y_p\)：永久收入
•\(r_b - r_m\)：债券与货币收益差
•\(r_e - r_m\)：股票与货币收益差
•\(\pi^e\)：预期通胀率
•\(w\)：非人力财富占比
弗里德曼的货币主义结论
•货币需求稳定且可预测
•货币供给影响名义变量，长期影响价格水平
•通胀归根结底是货币现象

通货膨胀的成因

需求拉动型通胀

总需求过度增长，经济过热。

财政赤字货币化
机制: 政府印钱支出
案例: 魏玛德国、委内瑞拉
信贷扩张
机制: 银行过度放贷
案例: 2008年前美国
外汇流入
机制: 央行被动投放基础货币
案例: 2000年代中国
消费热潮
机制: 需求超过产能
案例: 2021年美国
成本推动型通胀

供给侧成本上升推高价格。

工资上涨
传导机制: 劳动成本上升
案例: 1970s美国
原材料涨价
传导机制: 进口成本上升
案例: 石油危机
汇率贬值
传导机制: 进口品价格上涨
案例: 新兴市场
供应链中断
传导机制: 短缺抬价
案例: 2021-22疫情
结构性通胀
•部门间生产率差异
•工资"攀比"效应
•服务业通胀高于制造业（巴拉萨-萨缪尔森效应）
通胀预期

自我实现的预言——预期通胀会导致实际通胀。

\[\pi = \pi^e + 供需缺口因素\]
预期形成机制
适应性预期
公式: \(\pi^e_t = \pi_{t-1}\)
特点: 后视，调整缓慢
理性预期
公式: \(\pi^e_t = E[\pi_t
特点: I_{t-1}]\)
: 前瞻，利用所有信息
锚定预期
公式: \(\pi^e = \pi^{target}\)
特点: 央行可信度高时

通胀的成本与收益

通胀成本
菜单成本
描述: 调整价格的成本（印刷、沟通）
经济含义: 资源浪费
皮鞋成本
描述: 减少现金持有、频繁取款
经济含义: 时间浪费
税收扭曲
描述: 名义收益被征税，实际税负上升
经济含义: 资本形成受抑制
财富再分配
描述: 债权人损失，债务人受益
经济含义: 不公平
相对价格扭曲
描述: 价格信号失真
经济含义: 资源错配
不确定性
描述: 难以规划长期投资
经济含义: 增长受损
适度通胀的好处
调整实际工资
机制: 名义工资刚性下变相降薪
政策含义: 避免失业
避免零利率下限
机制: 保留货币政策空间
政策含义: 政策灵活性
减轻债务负担
机制: 通胀侵蚀实际债务
政策含义: 去杠杆工具
鼓励消费和投资
机制: 持币成本上升
政策含义: 刺激经济

为什么央行普遍选择2%通胀目标？

•足够低：不扭曲决策
•足够高：留有政策空间
•容易沟通：简单明确
•国际惯例：便于协调

关键模型与公式

1

模型1：费雪方程式与货币数量论

交易方程式
\[MV = PY\]
增长率形式
\[\frac{\Delta M}{M} + \frac{\Delta V}{V} = \frac{\Delta P}{P} + \frac{\Delta Y}{Y}\]

长期近似（假设V稳定）：

\[\pi \approx g_M - g_Y\]

货币增速减去实际增长率等于通胀率。

货币中性与超中性
•货币中性：货币供给变化只影响名义变量，不影响实际变量
•货币超中性：货币增长率变化也不影响实际变量
2

模型2：费雪效应

\[i = r + \pi^e\]
事前 vs 事后
事前实际利率
公式: \(r = i - \pi^e\)
可观察性: 不可直接观察
事后实际利率
公式: \(r_{ex-post} = i - \pi_{实际}\)
可观察性: 可计算
完全费雪效应
•预期通胀上升1%，名义利率上升1%
•实际利率不变
实证发现
•短期：费雪效应不完全（名义利率调整不完全）
•长期：费雪效应基本成立
3

模型3：泰勒规则

\[i_t = r^* + \pi_t + 0.5(\pi_t - \pi^*) + 0.5(y_t - y^*)\]
分解
\(r^*\)
含义: 自然实际利率
系数: 约2%
\(\pi_t\)
含义: 当期通胀
系数: 1（费雪效应）
\(\pi_t - \pi^*\)
含义: 通胀缺口
系数: 0.5（泰勒建议）
\(y_t - y^*\)
含义: 产出缺口
系数: 0.5（泰勒建议）
政策含义
•通胀高于目标：加息（且幅度大于通胀上升幅度）
•产出低于潜在：降息
泰勒原则
\[\frac{\partial i}{\partial \pi} > 1\]

对通胀的反应系数大于1，否则实际利率下降，刺激需求，通胀失控。

4

模型4：菲利普斯曲线

原始形式（1958）
\[\frac{\Delta W}{W} = -\alpha(u - u^*)\]

工资增长与失业率负相关。

预期增强形式
\[\pi = \pi^e - \alpha(u - u^*) + \epsilon\]
新凯恩斯菲利普斯曲线
\[\pi_t = \beta E_t[\pi_{t+1}] + \kappa \tilde{y}_t\]

通胀取决于预期通胀和产出缺口。

短期 vs 长期
短期
曲线形状: 向下倾斜
政策含义: 可以用通胀换取就业
长期
曲线形状: 垂直于自然率
政策含义: 通胀-失业无权衡
曲线移动
•通胀预期上升 → 曲线上移
•供给冲击（如油价）→ 曲线上移
•曲线平坦化：通胀对产出缺口敏感性下降（全球化效应？）

实务应用

美

案例1：美联储2022年加息周期

背景：2022年美国通胀飙升至9.1%（6月），为40年新高。

原因分析
财政刺激
贡献: 约30%
性质: 需求拉动
供应链中断
贡献: 约25%
性质: 成本推动
能源危机（俄乌）
贡献: 约20%
性质: 供给冲击
劳动力短缺
贡献: 约15%
性质: 工资推动
租金上涨
贡献: 约10%
性质: 结构性
美联储应对
2022年3月：开始加息（0.25%）
2022年5月-7月：激进加息（0.5%, 0.75%, 0.75%）
2023年7月：利率峰值（5.25-5.50%）
2024年：开始降息讨论
政策反思
1.为什么美联储最初误判通胀为"暂时性"（transitory）？
•供给冲击通常是暂时的
•疫情前低通胀环境形成思维定式
•模型基于锚定预期假设
2.延迟加息的后果：
•通胀预期开始松动
•需要更激进的加息来"追赶"
•金融市场剧烈波动
3.启示：
•央行信誉是通胀管理的核心资产
•供给冲击也可能触发通胀预期脱锚
•"宁可过度反应，不可反应不足"

案例2：日本的长期通缩与突破

背景：日本经历"失去的三十年"，长期通缩或低通胀。

通缩螺旋机制
资产泡沫破裂 → 财富缩水 → 需求不足 → 价格下跌
      ↓                                    ↓
 债务负担加重 ← 实际利率上升 ← 通缩预期形成 ←
政策尝试
1990s
政策: 传统降息
效果: 利率降至零
2000s
政策: 量化宽松（QE）
效果: 效果有限
2013
政策: 安倍经济学（QQE）
效果: 通胀短暂回升
2016
政策: 负利率 + YCC
效果: 稳定通胀预期
2022-23
政策: 全球通胀传导
效果: 首次持续通胀
2024年日本通胀突破
•输入性通胀（日元贬值、能源价格）
•工资-价格良性循环开始形成
•日本央行退出负利率（2024年3月）
启示
1.通缩预期一旦形成，极难打破
2.零利率下限严重约束货币政策
3.财政政策配合可能更有效
4.外部冲击可能帮助打破均衡
中

案例3：中国的货币政策框架转型

从数量型到价格型调控
中间目标
传统框架: M2增速
新框架: 利率
政策利率
传统框架: 存贷款基准利率
新框架: OMO利率 + MLF利率
传导机制
传统框架: 信贷额度
新框架: LPR → 贷款利率
操作方式
传统框架: 数量调控为主
新框架: 价格调控为主
LPR改革（2019年8月）
MLF利率 → 银行加点 → LPR → 贷款利率
  ↓
央行政策利率    市场化形成    实体经济融资成本
中国货币政策的特殊约束
汇率
表现: 美元加息周期中的贬值压力
应对: 容忍一定贬值
房地产
表现: 降息可能刺激房价
应对: 结构性工具
银行息差
表现: 降息压缩净息差

1.转型是渐进过程，需要配套改革

核心问题

概念理解类
1.为什么"钱印多了会通胀"？货币数量论的逻辑是什么？在什么条件下这一关系不成立？
•思考方向：MV=PY、货币流通速度变化、流动性陷阱
2.通胀对债权人和债务人有什么不同影响？这种财富再分配是否公平？
•思考方向：实际利率、通胀意外、名义债务
3.什么是"滞胀"？为什么它对政策制定者是噩梦？应该如何应对？
•思考方向：供给冲击、菲利普斯曲线、政策两难
分析应用类
4.如果预期通胀上升，你应该如何调整投资组合？哪些资产是通胀对冲工具？
•思考方向：实物资产、TIPS、股票、黄金、房地产
5.央行独立性为什么对控制通胀重要？政治干预货币政策会带来什么后果？
•思考方向：时间不一致性、通胀偏差、信誉
6.中国M2/GDP比例远高于其他国家，为什么没有引发高通胀？
•思考方向：货币流通速度、房地产吸纳、储蓄率
批判思维类
7.数字货币（CBDC）会如何改变货币政策的传导机制？央行能否实施"直升机撒钱"？
•思考方向：现金消失、负利率可行性、财政-货币边界
8.现代货币理论（MMT）认为政府可以无限印钱，你如何评价？什么情况下会失败？
•思考方向：通胀约束、汇率、资源限制
9.2%的通胀目标是否还合适？有人主张提高到3-4%，有人主张降到0%，你怎么看？
•思考方向：政策空间、通胀成本、零利率下限
