# 金融学术殿堂 - Finance Academy

一个现代化的金融教学Web应用，集成48门精品课程，提供交互式学习体验。

## 功能特性

- **48门精品课程**：涵盖宏观经济学、微观经济学、公司金融、资产定价、计量经济学、人工智能金融六大领域
- **交互式学习**：96+个交互组件，包括计算器、图表模拟器、流程图等
- **知识图谱**：每门课程配备Canvas知识图谱，可视化知识结构
- **学习进度追踪**：自动保存学习进度，可视化展示完成情况
- **响应式设计**：完美适配桌面端和移动端

## 课程结构

| 领域 | 课程数 | 核心内容 |
|------|--------|----------|
| 宏观经济学 | 8门 | GDP、通胀、货币政策、经济增长 |
| 微观经济学 | 8门 | 供需理论、市场结构、博弈论 |
| 公司金融 | 8门 | 财务分析、投融资决策、资本结构 |
| 资产定价 | 8门 | CAPM、多因子模型、衍生品定价 |
| 计量经济学 | 8门 | 回归分析、因果推断、时间序列 |
| 人工智能金融 | 8门 | 大模型应用、RAG、NLP分析 |

## 快速开始

### 方式一：直接打开
1. 使用现代浏览器（Chrome/Firefox/Safari/Edge）
2. 直接打开 `index.html` 文件
3. 开始浏览课程

### 方式二：本地服务器（推荐）
```bash
# 进入项目目录
cd finance-academy-web

# 使用Python启动服务器
python3 -m http.server 8080

# 或使用Node.js
npx serve .

# 然后访问 http://localhost:8080
```

## 使用指南

### 浏览课程
1. 首页展示六大知识领域概览
2. 点击"课程仪表盘"查看所有48门课程
3. 使用筛选器按领域浏览

### 学习课程
1. 点击课程卡片打开详情页
2. 查看学习目标、交互组件预览
3. 点击"打开完整课件"查看详细内容
4. 使用"知识图谱"查看知识结构（需在Obsidian中打开.canvas文件）

### 追踪进度
1. 进入"进度"页面查看总体完成情况
2. 各领域的学习进度一目了然
3. 进度自动保存到浏览器本地存储
