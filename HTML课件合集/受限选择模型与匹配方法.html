核心概念

二值选择模型

问题背景

因变量\(Y\)只取0或1两个值时，如何建模？

线性概率模型（LPM）的问题
\[P(Y=1|X) = X'\beta\]
预测值超出[0,1]
描述: 概率可能为负或大于1
异方差
描述: \(Var(Y
: X) = P(1-P)\)，随X变化
边际效应恒定
描述: 假设不合理
Logit模型
\[P(Y=1|X) = \Lambda(X'\beta) = \frac{e^{X'\beta}}{1+e^{X'\beta}} = \frac{1}{1+e^{-X'\beta}}\]
Probit模型
\[P(Y=1|X) = \Phi(X'\beta)\]

其中\(\Phi(\cdot)\)为标准正态分布的累积分布函数。

潜变量解释

存在不可观测的潜变量\(Y^*\)：

\[Y^* = X'\beta + \epsilon\]
\[Y = \begin{cases} 1 & \text{if } Y^* > 0 \\ 0 & \text{if } Y^* \leq 0 \end{cases}\]
•Logit：\(\epsilon\)服从Logistic分布
•Probit：\(\epsilon\)服从标准正态分布

边际效应

Logit的边际效应
\[\frac{\partial P(Y=1|X)}{\partial X_j} = \Lambda(X'\beta)(1-\Lambda(X'\beta)) \cdot \beta_j\]
Probit的边际效应
\[\frac{\partial P(Y=1|X)}{\partial X_j} = \phi(X'\beta) \cdot \beta_j\]
关键特点
•边际效应随\(X\)变化
•\(X'\beta\)接近0时边际效应最大
•\(X'\beta\)远离0时边际效应趋近0
边际效应的计算方式
均值处边际效应（MEM）
计算: 在\(X=\bar{X}\)处计算
特点: 可能不代表任何真实个体
平均边际效应（AME）
计算: 每个观测值处计算后平均
特点: 更常用，解释更直观
代表个体边际效应
计算: 在特定X值处计算
特点: 针对特定情景

最大似然估计

似然函数
\[L(\beta) = \prod_{i=1}^{n} P(Y_i|X_i)^{Y_i} (1-P(Y_i|X_i))^{1-Y_i}\]
对数似然
\[\ell(\beta) = \sum_{i=1}^{n} [Y_i \ln P_i + (1-Y_i) \ln(1-P_i)]\]
一阶条件
\[\frac{\partial \ell}{\partial \beta} = \sum_{i=1}^{n} (Y_i - P_i) \cdot f_i \cdot X_i = 0\]

其中\(f_i\)为概率密度函数。

标准误
\[\widehat{Var}(\hat{\beta}) = \left[-\frac{\partial^2 \ell}{\partial \beta \partial \beta'}\right]^{-1}\]

模型评价

拟合优度指标
对数似然
公式: \(\ell(\hat{\beta})\)
解释: 越大越好
伪R²
公式: \(1 - \ell(\hat{\beta})/\ell(0)\)
解释: 类比R²，但解释不同
AIC
公式: \(-2\ell + 2k\)
解释: 模型比较
BIC
公式: \(-2\ell + k\ln n\)
解释: 更惩罚参数
预测能力评价
混淆矩阵
实际=1
预测=1: TP
预测=0: FN
实际=0
预测=1: FP
预测=0: TN
评价指标
•准确率：\((TP+TN)/(TP+TN+FP+FN)\)
•灵敏度（召回率）：\(TP/(TP+FN)\)
•特异度：\(TN/(TN+FP)\)
•精确率：\(TP/(TP+FP)\)
ROC曲线与AUC
•ROC：灵敏度 vs (1-特异度)
•AUC：曲线下面积，0.5为随机，1为完美

多项选择模型

多项Logit模型（无序选择）：

\[P(Y=j|X) = \frac{e^{X'\beta_j}}{\sum_{k=0}^{J} e^{X'\beta_k}}\]

归一化：\(\beta_0 = 0\)

独立不相关备选项假设（IIA）
\[\frac{P(Y=j)}{P(Y=k)} = e^{X'(\beta_j - \beta_k)}\]

不依赖于其他选项，可能不合理（红蓝公交车问题）。

有序Probit/Logit模型

潜变量：\(Y^* = X'\beta + \epsilon\)

\[Y = j \quad \text{if} \quad c_{j-1} < Y^* \leq c_j\]
\[P(Y=j|X) = F(c_j - X'\beta) - F(c_{j-1} - X'\beta)\]

适用于有序分类变量（如信用评级、满意度）。

截断与删失数据

截断（Truncation）

只观察到满足某条件的样本。

例：只研究成功上市的公司。

删失（Censoring）

知道样本存在但不知道具体值。

例：收入调查中高收入者填写"100万以上"。

Tobit模型（Type I Tobit）：

潜变量：\(Y^* = X'\beta + \epsilon, \quad \epsilon \sim N(0, \sigma^2)\)

观测值：

\[Y = \begin{cases} Y^* & \text{if } Y^* > 0 \\ 0 & \text{if } Y^* \leq 0 \end{cases}\]
似然函数
\[L = \prod_{Y_i=0} P(Y^*_i \leq 0) \cdot \prod_{Y_i>0} f(Y_i|X_i)\]
\[= \prod_{Y_i=0} \Phi\left(-\frac{X_i'\beta}{\sigma}\right) \cdot \prod_{Y_i>0} \frac{1}{\sigma}\phi\left(\frac{Y_i - X_i'\beta}{\sigma}\right)\]
边际效应
对\(E[Y^*
公式: X]\)
含义: \(\beta_j\)
: 潜变量效应
对\(E[Y
公式: X, Y>0]\)
含义: \(\beta_j \cdot (1 - \lambda(\alpha)[\lambda(\alpha) - \alpha])\)
: 未删失样本效应
对\(P(Y>0
公式: X)\)
含义: \(\beta_j \cdot \phi(\alpha)/\sigma\)
: 参与概率效应
对\(E[Y
公式: X]\)
含义: \(\beta_j \cdot \Phi(\alpha)\)
: 无条件期望效应

其中\(\alpha = X'\beta/\sigma\)，\(\lambda(\alpha) = \phi(\alpha)/\Phi(\alpha)\)（逆米尔斯比率）。

样本选择模型

问题设定

选择方程：

\[S^* = Z'\gamma + u\]
\[S = \mathbf{1}(S^* > 0)\]

结果方程（仅在\(S=1\)时观测）：

\[Y = X'\beta + \epsilon\]

如果\(Corr(u, \epsilon) \neq 0\)，直接用\(S=1\)样本估计会有偏。

Heckman两步法

第一步：Probit估计选择方程

\[\hat{P}(S=1|Z) = \Phi(Z'\hat{\gamma})\]

计算逆米尔斯比率：

\[\hat{\lambda}_i = \frac{\phi(Z_i'\hat{\gamma})}{\Phi(Z_i'\hat{\gamma})}\]

第二步：在\(S=1\)样本中加入\(\hat{\lambda}\)回归

\[Y_i = X_i'\beta + \rho\sigma_\epsilon \hat{\lambda}_i + \eta_i\]

\(\hat{\lambda}\)系数显著 → 存在样本选择偏差。

识别条件
•排除性约束：\(Z\)中有不在\(X\)中的变量
•否则仅靠函数形式识别（不稳健）

关键公式与估计方法

关键问题解答

概念类问题

Q1: Logit系数的odds ratio如何解释？

A:

Odds Ratio定义
\[OR = e^{\beta}\]
解释
•\(OR > 1\)：X增加1单位，发生的几率增加\((OR-1)\times 100\%\)
•\(OR < 1\)：X增加1单位，发生的几率降低\((1-OR)\times 100\%\)
•\(OR = 1\)：X对发生几率无影响
示例

若\(\beta_{Income} = -0.5\)，则\(OR = e^{-0.5} = 0.61\)

收入每增加1单位，违约几率降低39%。

注意
•Odds ≠ 概率
•\(Odds = P/(1-P)\)
•当事件罕见时，OR近似相对风险

Q2: 什么时候应该用Tobit而不是OLS？

A:

使用Tobit的情况
1.因变量有大量观测值聚集在边界（如0）
2.边界值是真实的限制，不是缺失
3.关心潜变量\(Y^*\)的效应
OLS的问题
•忽略删失导致的选择偏差
•系数向下偏误
•预测可能超出合理范围
判断标准
•边界观测占比高（如>10%）
•理论上存在潜变量解释
•关心"是否"和"多少"两个决策

Q3: Heckman选择模型中的排除性约束是什么？为什么重要？

A:

排除性约束

选择方程中的某些变量不出现在结果方程中。

数学表示
•选择方程：\(S = \mathbf{1}(Z'\gamma > 0)\)
•结果方程：\(Y = X'\beta + \epsilon\)
•要求：\(Z\)中有不在\(X\)中的变量
重要性
1.识别：没有排除性约束，模型仅靠函数形式识别
2.稳定性：函数形式识别对模型假设敏感
3.可信度：排除变量有经济学含义更可信
寻找排除变量的原则
•影响选择但不直接影响结果
•理论支持
•例：地理距离影响是否参与，但不影响结果
应用类问题

Q5: 多项Logit的IIA假设如何检验？违反时怎么办？

A:

IIA假设

删除一个选项不改变其余选项的相对概率。

检验方法
1.Hausman-McFadden检验
•删除一个选项重新估计
•比较系数是否显著不同
2.Small-Hsiao检验
•类似但更稳健
违反IIA时的替代
Nested Logit
特点: 允许组内相关
适用情况: 有层级结构
Mixed Logit
特点: 允许系数随机
适用情况: 偏好异质性
Multinomial Probit
特点: 完全灵活协方差
适用情况: 计算复杂

Q6: 如何处理信用评分模型中的类别不平衡问题？

A:

问题

违约样本通常只占1-5%，模型倾向预测"不违约"。

解决方案
过采样
描述: 复制少数类样本
优缺点: 简单但可能过拟合
SMOTE
描述: 合成少数类样本
优缺点: 更平滑
欠采样
描述: 减少多数类样本
优缺点: 损失信息
调整阈值
描述: 降低分类阈值
优缺点: 不改变模型
加权
描述: 给少数类更高权重
优缺点: 最常用
加权MLE
\[\ell(\beta) = \sum_{i: Y_i=1} w_1 \ln P_i + \sum_{i: Y_i=0} w_0 \ln(1-P_i)\]

设置\(w_1/w_0\)等于不平衡比例的倒数。

批判性问题

Q7: Heckman模型的局限性是什么？

A:

主要局限
1.识别困难
•需要有效的排除变量
•仅靠函数形式识别不可靠
2.分布假设敏感
•假设正态分布
•违反时结果不一致
3.两步法效率损失
•标准误需要调整
•MLE更有效但计算复杂
4.非参数替代
•控制函数方法
•边界方法
建议
•报告排除变量的合理性
•进行敏感性分析
•与其他方法比较

Q8: 为什么金融中普遍使用Logit而非Probit？

A:

实践中Logit更常用的原因
1.Odds Ratio解释直观
•金融从业者熟悉
•监管报告常用
2.计算简便
•有闭式解
•不需数值积分
3.厚尾更合适
•金融数据极端值多
•Logistic尾部更厚
4.历史惯例
•信用评分行业标准
•便于比较
实际差异
•中间概率区域差异很小
•极端概率区域Logit更保守
•系数可转换：\(\beta_{Logit} \approx 1.6 \times \beta_{Probit}\)

Q9: 样本选择偏差与内生性有什么关系？

A:

联系与区别
来源
样本选择偏差: 样本非随机选择
一般内生性: 遗漏变量、测量误差等
表现
样本选择偏差: 只观察到部分样本
一般内生性: 观察到全部但X与ε相关
偏差性质
样本选择偏差: 有偏且不一致
一般内生性: 有偏且不一致
解决方法
样本选择偏差: Heckman选择模型
一般内生性: IV、DID、RDD等
共同本质

都是因为\(E[\epsilon|X] \neq 0\)

区别
•样本选择：因为样本筛选机制
•内生性：因为变量本身的问题
同时存在时

需要同时处理，如使用控制函数方法。
