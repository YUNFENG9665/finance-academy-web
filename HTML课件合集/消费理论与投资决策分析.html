核心概念

消费函数的演进

凯恩斯消费函数（1936）
\[C = \bar{C} + cY\]
•\(\bar{C}\)：自主消费（与收入无关的基本消费）
•\(c\)：边际消费倾向（MPC），\(0 < c < 1\)
•\(Y\)：当期可支配收入
凯恩斯三大心理法则
消费随收入增加
内容: 收入↑ → 消费↑
数学表达: \(\frac{dC}{dY} > 0\)
MPC < 1
内容: 消费增加 < 收入增加
数学表达: \(0 < c < 1\)
APC递减
内容: 富人平均消费倾向低
数学表达: \(\frac{dAPC}{dY} < 0\)
边际消费倾向 vs 平均消费倾向
\[MPC = \frac{dC}{dY} = c\]
\[APC = \frac{C}{Y} = \frac{\bar{C}}{Y} + c\]
凯恩斯消费函数的实证问题
短期 vs 长期
表现: 短期APC递减，长期APC稳定
说明: Kuznets悖论
消费平滑
表现: 消费比收入更稳定
说明: 凯恩斯函数无法解释

生命周期假说（Modigliani, 1954）

核心思想：消费者在整个生命周期内平滑消费。

基本模型
\[C = \frac{1}{T}(W + R \cdot Y)\]
•\(W\)：初始财富
•\(R\)：剩余工作年限
•\(Y\)：年收入
•\(T\)：剩余寿命
更一般的形式
\[C = \alpha W + \beta Y\]
•\(\alpha\)：财富的边际消费倾向
•\(\beta\)：收入的边际消费倾向
生命周期消费路径
收入/消费
    │       ╱╲ 收入
    │      ╱  ╲
    │─────/────\──────── 消费（平滑）
    │    ╱      ╲
    │   ╱        ╲
    └──────────────────→ 年龄
       工作期    退休期
       储蓄      消耗储蓄
关键洞见
消费取决于终身资源
含义: 不只是当期收入
政策影响: 暂时性减税效果有限
财富效应
含义: 股市/房市影响消费
政策影响: 资产价格波动传导
储蓄生命周期
含义: 工作期储蓄，退休期消耗
政策影响: 人口老龄化影响储蓄率
遗产动机
含义: 可能延长积累期
政策影响: 财富不平等代际传递

永久收入假说（Friedman, 1957）

核心思想：消费取决于永久收入而非暂时收入。

\[C = kY_P\]
•\(k\)：消费比例（接近1）
•\(Y_P\)：永久收入（长期平均预期收入）
收入分解
\[Y = Y_P + Y_T\]
•\(Y_P\)：永久收入（持久性、可预期）
•\(Y_T\)：暂时收入（随机波动、均值为零）
永久收入的形成
\[Y_P^t = \lambda Y^t + (1-\lambda)Y_P^{t-1}\]

适应性预期：永久收入是过去收入的加权平均。

政策含义
暂时性减税
收入性质: 暂时收入
消费反应: 小（储蓄为主）
永久性减税
收入性质: 永久收入
消费反应: 大（消费增加）
一次性转移支付
收入性质: 暂时收入
消费反应: 小
生命周期 vs 永久收入
时间范围
生命周期假说: 有限生命
永久收入假说: 无限期（可含遗产）
财富作用
生命周期假说: 明确纳入
永久收入假说: 隐含在永久收入中
关注点
生命周期假说: 储蓄-消耗周期
永久收入假说: 收入波动的平滑
共同点
生命周期假说: 都强调前瞻性、跨期平滑
永久收入假说:

随机游走假说（Hall, 1978）

理性预期下的消费行为

如果消费者是理性的、前瞻性的：

\[C_{t+1} = C_t + \epsilon_{t+1}\]

消费是随机游走，不可预测！

推导（欧拉方程）：

\[u'(C_t) = \beta(1+r)E_t[u'(C_{t+1})]\]

如果\(\beta(1+r) = 1\)（不耐心程度=利率），则\(E_t[C_{t+1}] = C_t\)

含义
•消费变化只反映新信息
•可预测的收入变化不应影响消费
•只有收入"意外"才改变消费
实证检验
消费对滞后收入有反应
是否支持理论: 不支持
可能解释: 流动性约束
消费对可预测收入变化有反应
是否支持理论: 不支持
可能解释: 有限理性
消费对退税反应过大
是否支持理论: 不支持
可能解释: 心理账户
理论失败的原因
1.流动性约束：部分消费者无法借贷
2.有限理性：认知能力有限，采用经验法则
3.预防性储蓄：面对不确定性额外储蓄
4.心理账户：不同收入来源不可替代

投资理论

投资的决定因素
因素影响方向机制弹性估计
实际利率负提高资本成本-0.5至-1
预期需求正产能需求（加速数）高
不确定性负推迟投资（实物期权）显著
企业利润正内部融资来源中等
税收政策取决于政策影响税后回报显著
信贷条件正融资可得性经济衰退时大
投资的组成
类型内容占比（美国）特点
设备投资机器、车辆~45%波动大
建筑投资厂房、办公楼~25%周期长
住宅投资住房建设~25%利率敏感
存货投资库存变化~5%高度波动

加速数原理（Accelerator）：

\[I_t = v(Y_t - Y_{t-1})\]

投资取决于产出变化，\(v\)是资本-产出比。

问题：只考虑需求，忽略资本成本。

新古典投资理论（Jorgenson）
\[I = f(r, MPK, \delta)\]
\[MPK = r + \delta\]

最优资本存量使得资本边际产品等于资本使用成本。

托宾Q理论
\[Q = \frac{企业市场价值}{资本重置成本}\]
Q > 1
含义: 市场高估资本价值
投资决策: 增加投资（新建>买现有）
Q < 1
含义: 市场低估资本价值
投资决策: 减少投资（买现有>新建）
Q = 1
含义: 边际状态
投资决策: 投资=折旧
实物期权理论

不确定性下，等待获取信息有价值。

\[投资触发点 > 传统NPV=0的点\]
不可逆投资的特点
•资本一旦安装难以撤回
•等待期权价值
•不确定性抑制投资

关键模型与公式

1

模型1：乘数效应

政府支出乘数
\[\Delta Y = \frac{1}{1-c} \Delta G = k_G \cdot \Delta G\]
\[k_G = \frac{1}{1-MPC}\]
税收乘数
\[k_T = \frac{-MPC}{1-MPC}\]
平衡预算乘数
\[k_{BB} = k_G + k_T = \frac{1-MPC}{1-MPC} = 1\]

数值示例（MPC = 0.8）：

政府支出
公式: 1/(1-0.8)
数值: 5
税收
公式: -0.8/(1-0.8)
数值: -4
平衡预算
公式: —
数值: 1
开放经济中的乘数
\[k = \frac{1}{1 - c(1-t) + m}\]
•\(t\)：边际税率
•\(m\)：边际进口倾向

开放经济乘数更小（漏出更多）。

2

模型2：跨期消费选择

两期模型
\[\max U(C_1, C_2) \quad s.t. \quad C_1 + \frac{C_2}{1+r} = Y_1 + \frac{Y_2}{1+r}\]

预算约束：现值形式

\[PV(消费) = PV(收入) = 终身财富\]
欧拉方程
\[\frac{U'(C_1)}{U'(C_2)} = \beta(1+r)\]
CRRA效用函数下
\[\frac{C_2}{C_1} = [\beta(1+r)]^\sigma\]
•\(\sigma = 1/\rho\)：跨期替代弹性
•\(\rho\)：相对风险厌恶系数
利率变化的效应
替代效应
方向: r↑ → C1↓, C2↑
机制: 未来消费更便宜
收入效应
方向: r↑ → C1↑, C2↑（净储蓄者）
机制: 利息收入增加
净效应
方向: 不确定
机制: 取决于初始位置和偏好
3

模型3：托宾Q投资模型

边际Q定义
\[q = \frac{多一单位资本的市场价值}{多一单位资本的重置成本}\]

平均Q定义（实际可测）：

\[Q = \frac{股权市值 + 债务市值}{资本重置成本}\]
投资方程
\[\frac{I}{K} = f(q), \quad f'(q) > 0\]
与股市的联系
•股价上涨 → Q上升 → 投资增加
•金融市场与实体经济的桥梁
实证问题
解释力有限
表现: 现金流比Q更显著
可能原因: 融资约束
平均Q≠边际q
表现: 市场泡沫、调整成本
可能原因: 测量问题
4

模型4：资本使用者成本

\[UC = P_K(r + \delta - \frac{\Delta P_K}{P_K})\]
•\(P_K\)：资本品价格
•\(r\)：实际利率
•\(\delta\)：折旧率
•\(\frac{\Delta P_K}{P_K}\)：资本增值率
含税的资本使用者成本
\[UC = \frac{P_K}{1-\tau}[(1-\tau z)(r+\delta) - \frac{\Delta P_K}{P_K}]\]
•\(\tau\)：公司税率
•\(z\)：折旧税盾现值
最优资本存量条件
\[MPK = UC\]

案例1：中国高储蓄率之谜

背景：中国储蓄率长期高达40-50%，远超全球平均（约25%）。

储蓄率国际比较
中国
储蓄率（2023）: ~45%
特点: 全球最高之一
美国
储蓄率（2023）: ~18%
特点: 消费主导
日本
储蓄率（2023）: ~28%
特点: 老龄化中下降
印度
储蓄率（2023）: ~30%
特点: 发展中国家典型

原因分析（生命周期视角）：

预防性储蓄
机制: 社保不完善、医疗教育支出大
贡献估计: ~30%
人口结构
机制: 劳动年龄人口占比高
贡献估计: ~25%
房价因素
机制: 购房首付需要大量储蓄
贡献估计: ~20%
文化因素
机制: 储蓄传统、面子消费
贡献估计: ~15%
收入分配
机制: 高收入群体储蓄率高
贡献估计: ~10%
企业储蓄的重要性
•中国企业储蓄占总储蓄约40%
•原因：利润留存、分红少、融资渠道窄

高储蓄率是好事还是坏事？

增长
好处: 资本积累快
坏处: 消费不足、内需弱
稳定
好处: 外部冲击缓冲
坏处: 全球失衡
福利
好处: —
坏处: 当代人福利牺牲
政策方向
1.完善社会保障 → 减少预防性储蓄
2.提高劳动收入占比 → 增加消费能力
3.发展消费金融 → 平滑消费
4.租购并举 → 减少购房储蓄压力

案例3：企业投资的不确定性效应

背景：政策不确定性抑制企业投资。

实证发现（Baker, Bloom & Davis不确定性指数）：

2016美国大选
不确定性指数变化: +40%
投资反应: 投资延迟
2018-19贸易战
不确定性指数变化: +80%
投资反应: 资本支出下降
2020疫情
不确定性指数变化: +200%
投资反应: 投资骤降
不确定性影响投资的渠道
实物期权
机制: 等待获取信息
理论基础: 不可逆投资
融资约束
机制: 风险溢价上升
理论基础: 代理成本
需求不确定
机制: 预期需求下降
理论基础: 加速数原理
管理行为
机制: 风险厌恶加剧
理论基础: 行为公司金融
政策含义
1.政策一致性和可预测性本身有价值
2.不确定性是经济周期放大器
3.央行沟通策略的重要性（前瞻指引）
中国制造业投资案例
•房地产调控不确定性 → 相关产业投资延迟
•平台经济监管 → 科技企业资本支出下降
•地缘政治 → 供应链投资重构

核心问题

概念理解类
1.凯恩斯消费函数与生命周期/永久收入假说有什么本质区别？哪个更符合现实？
•思考方向：前瞻性、时间范围、收入分类
2.为什么暂时性减税刺激消费的效果可能不大？实际政策中如何设计才能更有效？
•思考方向：永久收入、流动性约束、心理账户
3.托宾Q理论如何将金融市场与实体投资联系起来？为什么股市繁荣可能刺激投资？
•思考方向：市场估值、融资成本、信心效应
分析应用类
4.如果利率上升，消费会如何变化？需要考虑哪些效应？不同群体反应相同吗？
•思考方向：替代效应、收入效应、净储蓄者vs净借款者
5.政府应该用减税还是增加支出来刺激经济？乘数有何不同？考虑什么因素？
•思考方向：乘数大小、挤出效应、政策时滞
6.如何解释中国储蓄率高而消费率低？这对经济再平衡意味着什么？
•思考方向：预防性储蓄、房价、社保、分配
批判思维类
7.消费者真的能按生命周期理论那样理性规划吗？行为经济学（如心理账户、双曲贴现）提出了什么挑战？
•思考方向：有限理性、自我控制、nudge政策
8.在零利率环境下，货币政策如何影响投资？传统渠道失效后有什么替代？
•思考方向：QE、前瞻指引、财政-货币配合
9.李嘉图等价说减税不会刺激消费（因为预期未来税收增加），这个理论在实践中成立吗？为什么？
•思考方向：完美预期、代际关系、流动性约束
