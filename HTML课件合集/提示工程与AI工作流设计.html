提示工程基础

什么是提示工程

提示工程（Prompt Engineering）是指设计和优化输入给大语言模型的文本（提示/Prompt），以获得期望输出的技术和方法。好的提示可以显著提升模型输出的质量、准确性和可用性。

好提示的核心要素
清晰的任务说明
说明: 明确告诉模型要做什么
示例: "分析以下财务报表，指出三个主要风险点"
具体的输出格式
说明: 指定期望的输出结构
示例: "以JSON格式输出，包含字段：公司名、评级、理由"
相关的上下文
说明: 提供必要的背景信息
示例: "这是一家科技行业的初创公司，成立于2020年"
示例（Few-shot）
说明: 提供输入输出示例
示例: "示例：输入A→输出B，现在请处理：输入C"
角色设定
说明: 给模型设定专业身份
示例: "你是一位拥有20年经验的财务分析师"
约束条件
说明: 设定限制和边界
示例: "回答不超过200字，不要推测未披露的信息"
提示结构模板
markdown
你是一位[专业角色]，擅长[专业领域]。

请执行以下任务：[具体任务描述]

以下是需要处理的数据：
{数据内容}

请按照以下格式输出：
- 格式：[JSON/表格/段落]
- 字段：[具体字段说明]
- 约束：[字数、风格等限制]

输入：[示例输入]
输出：[示例输出]

高级提示技术

思维链（Chain-of-Thought, CoT）

原理：引导模型像人类一样逐步推理，而不是直接给出答案。

基本形式
markdown
请一步步分析以下问题：
[问题内容]

请按以下步骤思考：
1. 首先，识别问题的关键信息
2. 然后，列出相关的计算步骤
3. 接着，逐步执行计算
4. 最后，给出结论并验证
金融应用示例
markdown
请分析以下公司的估值是否合理：
- 公司：某科技公司
- 市值：500亿
- 净利润：20亿
- 营收增速：30%
- 行业平均PE：25x

请一步步分析：
1. 计算当前PE倍数
2. 与行业平均对比
3. 考虑增速给予的溢价
4. 综合判断估值水平
5. 给出投资建议

自一致性（Self-Consistency）

原理：多次采样，选择最一致的答案。

实现方式
python
import openai

def self_consistent_analysis(prompt, n_samples=5):
    """自一致性分析：多次采样取共识"""
    responses = []
    for _ in range(n_samples):
        response = openai.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.7  # 保持一定随机性
        )
        responses.append(response.choices[0].message.content)

    # 投票选择最一致的答案
    return get_majority_answer(responses)

思维树（Tree-of-Thought, ToT）

原理：探索多个推理路径，评估并选择最优路径。

结构
问题
├── 思路A
│   ├── 子步骤A1 → 评估分数
│   └── 子步骤A2 → 评估分数
├── 思路B
│   ├── 子步骤B1 → 评估分数
│   └── 子步骤B2 → 评估分数
└── 选择最优路径 → 最终答案

**ReAct框架**（Reasoning + Acting）：

```python
class ReActAgent:
    """ReAct模式的Agent实现"""

    def __init__(self, llm, tools):
        self.llm = llm
        self.tools = tools
        self.memory = []

    def run(self, task):
        """执行任务"""
        self.memory.append(f"任务: {task}")

        while not self.is_done():
            # 思考：分析当前状态
            thought = self.think()
            self.memory.append(f"思考: {thought}")

            # 行动：选择并执行工具
            action, action_input = self.decide_action(thought)
            self.memory.append(f"行动: {action}({action_input})")

            # 观察：获取执行结果
            observation = self.execute_action(action, action_input)
            self.memory.append(f"观察: {observation}")

        return self.generate_final_answer()

from langchain.chat_models import ChatOpenAI
from langchain.chains import LLMChain, SequentialChain
from langchain.prompts import PromptTemplate

extract_prompt = PromptTemplate(
    input_variables=["raw_data"],
    template="""
    从以下财务数据中提取关键指标：
    {raw_data}

    请提取：营收、净利润、毛利率、ROE、资产负债率
    输出格式：JSON
    """
)
extract_chain = LLMChain(llm=llm, prompt=extract_prompt, output_key="metrics")

compare_prompt = PromptTemplate(
    input_variables=["metrics"],
    template="""
    基于以下指标进行同行业对比分析：
    {metrics}

    行业平均数据：
    - 毛利率：35%
    - ROE：15%
    - 资产负债率：50%

    请分析该公司在行业中的位置。
    """
)
compare_chain = LLMChain(llm=llm, prompt=compare_prompt, output_key="comparison")

report_prompt = PromptTemplate(
    input_variables=["metrics", "comparison"],
    template="""
    基于以下分析生成投资摘要：

    财务指标：{metrics}
    行业对比：{comparison}

    请生成300字以内的投资摘要，包括：优势、劣势、投资建议。
    """
)
report_chain = LLMChain(llm=llm, prompt=report_prompt, output_key="report")

analysis_workflow = SequentialChain(
    chains=[extract_chain, compare_chain, report_chain],
    input_variables=["raw_data"],
    output_variables=["metrics", "comparison", "report"]
)

result = analysis_workflow({"raw_data": financial_data})

适用场景: 迭代优化

人机协作
适用场景: 需要人工确认
示例: 重要决策前的审核

核心模式与公式

1

提示效果评估

评估维度
准确性
指标: Accuracy
计算方法: 正确回答数 / 总问题数
相关性
指标: Relevance
计算方法: 人工评分 1-5 或 LLM 评分
完整性
指标: Completeness
计算方法: 覆盖的要点数 / 应覆盖要点数
一致性
指标: Consistency
计算方法: 多次回答的相似度
有用性
指标: Usefulness
计算方法: 用户满意度评分
2

Agent性能指标

任务完成率
\[TCR = \frac{成功完成的任务数}{总任务数} \times 100\%\]
平均步骤数
\[AvgSteps = \frac{\sum_{i=1}^{n} steps_i}{n}\]
工具使用效率
\[ToolEfficiency = \frac{有效工具调用数}{总工具调用数}\]
3

工作流优化

延迟优化
•并行化独立步骤
•缓存中间结果
•使用更快的模型处理简单任务
成本优化
•任务路由：简单任务用小模型
•Prompt压缩：减少token数
•批处理：合并相似请求

def assess_risk_tolerance(answers):
    """评估客户风险承受能力"""
    score = 0

    # 投资期限
    if answers["investment_horizon"] > 5:
        score += 3
    elif answers["investment_horizon"] > 2:
        score += 2
    else:
        score += 1

    # 收入稳定性
    if answers["income_stability"] == "very_stable":
        score += 3
    elif answers["income_stability"] == "stable":
        score += 2
    else:
        score += 1

    # 损失容忍度
    if answers["max_loss_tolerance"] > 20:
        score += 3
    elif answers["max_loss_tolerance"] > 10:
        score += 2
    else:
        score += 1

    # 投资经验
    if answers["investment_experience"] == "experienced":
        score += 3
    elif answers["investment_experience"] == "moderate":
        score += 2
    else:
        score += 1

    # 确定风险等级
    if score >= 10:
        return "激进型"
    elif score >= 7:
        return "成长型"
    elif score >= 4:
        return "平衡型"
    else:
        return "保守型"

from jinja2 import Template

class PromptTemplateManager:
    """提示模板管理器"""

    def __init__(self, template_dir="prompts/"):
        self.template_dir = template_dir
        self.templates = {}
        self.load_templates()

    def load_templates(self):
        """加载所有模板"""
        for file in os.listdir(self.template_dir):
            if file.endswith(".jinja2"):
                name = file.replace(".jinja2", "")
                with open(os.path.join(self.template_dir, file)) as f:
                    self.templates[name] = Template(f.read())

    def render(self, template_name, **kwargs):
        """渲染模板"""
        if template_name not in self.templates:
            raise ValueError(f"Template {template_name} not found")
        return self.templates[template_name].render(**kwargs)

    def get_financial_analysis_prompt(self, company_data):
        """获取财务分析提示"""
        return self.render("financial_analysis",
                          data=company_data,
                          current_date=datetime.now().strftime("%Y-%m-%d"))

manager = PromptTemplateManager()
