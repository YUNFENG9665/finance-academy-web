## 更新方法（Git 自动部署）

课件有更新时，在终端执行以下命令，push 后 Netlify **自动部署**，无需手动操作：

```bash
cd "/Users/ethen/Documents/MAC/02-知识领域/99-金融学理论学习/finance-academy-web"

# 1. 同步最新课件
cp -r ../HTML课件合集/* HTML课件合集/

# 2. 提交并推送 → 自动触发 Netlify 部署
git add .
git commit -m "更新课件：描述本次改动"
git push
```

部署进度可在此查看：https://app.netlify.com/projects/finance-academy-web/deploys
