市场基本力量：供给与需求

市场价格不是由任何人"决定"的，而是由无数买卖双方的互动涌现出来的。

D 需求理论

消费者在各种可能的价格水平下愿意且能够购买的商品数量。

\( Q_d = f(P,\, I,\, P_s,\, P_c,\, T,\, E,\, N) \)
商品价格 \(P\) ↑需求量 ↓（沿曲线）
收入 \(I\) ↑正常品需求 ↑（右移）
替代品价格 \(P_s\) ↑本品需求 ↑
互补品价格 \(P_c\) ↑本品需求 ↓

S 供给理论

生产者在各种可能的价格水平下愿意且能够提供的商品数量。

\( Q_s = g(P,\, C,\, T,\, P_e,\, G,\, N) \)
商品价格 \(P\) ↑供给量 ↑（沿曲线）
生产成本 \(C\) ↑供给 ↓（左移）
技术水平 \(T\) ↑供给 ↑（右移）
预期价格 \(P_e\) ↑当前供给 ↓

价格机制：市场的"隐形指挥棒"

信息传递

价格上涨 = "有利可图"；价格下跌 = "调整产能"；价差 = "套利机会"

激励协调

高价激励生产者增加供给，低价激励消费者增加需求，引导资源最优配置

配置效率

P = MC 时社会福利最大。哈耶克：价格汇总分散在千百万个体中的知识

弹性：测量市场敏感度的标尺

当一个变量变化1%时，另一个变量变化多少？使用百分比消除单位影响，使不同商品可比。

需求价格弹性

\( E_d = \frac{dQ}{dP} \cdot \frac{P}{Q} \)
\(|E_d| > 1\) 富有弹性奢侈品、耐用品
\(|E_d| < 1\) 缺乏弹性必需品、成瘾品
\(|E_d| = 1\) 单位弹性总收益最大化点

需求收入弹性

\( E_I = \frac{\%\Delta Q_d}{\%\Delta I} \)
\(E_I > 1\) 奢侈品高端汽车、珠宝
\(0 < E_I < 1\) 必需品食品、基本服装
\(E_I < 0\) 劣等品方便面、二手商品

交叉价格弹性

\( E_{xy} = \frac{\%\Delta Q_x}{\%\Delta P_y} \)
\(E_{xy} > 0\) 替代品可口可乐 / 百事可乐
\(E_{xy} < 0\) 互补品汽车 / 汽油
\(E_{xy} = 0\) 无关商品面包 / 电脑

总收益与弹性关系

\( \frac{dTR}{dP} = Q(1 - |E_d|) \)
弹性需求降价总收益 ↑ 薄利多销
非弹性需求涨价总收益 ↑ 奇货可居
单位弹性总收益不变（已最大）

边际思维：经济学的第一性原理

经济学家不问"要不要"，而问"多一点还是少一点"。

边际概念体系

MU边际效用 — 多消费一单位的额外满足感
MR边际收益 — 多卖一单位的额外收入
MC边际成本 — 多生产一单位的额外成本
MP边际产出 — 多投入一单位要素的额外产出

最优决策规则

\( MR = MC \)

利润最大化的充要条件

+当 \(MR > MC\)：增加产量能增加利润
-当 \(MR < MC\)：减少产量能增加利润
=只有 \(MR = MC\) 时，利润达到最大

金融市场中的应用

将微观经济学原理映射到真实金融市场

股票市场

短期内已上市股票供给几乎固定，价格主要由需求驱动。

买方报价(Bid) / 卖方报价(Ask) = 成交价即"均衡"

利率 = 资金价格

资金需求（企业投资）与资金供给（储蓄）共同决定利率。

\( L_d(i) = L_s(i) \)
汇率 = 货币相对价格

购买力平价：长期汇率反映两国的相对购买力。

\( E = P_{domestic} / P_{foreign} \)
期权隐含波动率

期权价格由供需决定，隐含波动率反映市场预期。"波动率微笑"源于不同执行价的供需差异。

Beta = 收益弹性

Beta系数 = 股票收益对市场收益的"弹性"

\( \beta = \frac{Cov(R_i, R_m)}{Var(R_m)} \)
消费者/生产者剩余

完全竞争均衡时总剩余最大化：

\( CS = \int_0^{Q^*} P_d(Q)\,dQ - P^* Q^* \)

题目 1：求均衡价格、均衡数量与需求弹性

需求函数 \(Q_d = 200 - 4P\)，供给函数 \(Q_s = -40 + 6P\)

解答

(a) 令 \(Q_d = Q_s\)：\(200 - 4P = -40 + 6P\)，解得 \(P^* = 24\)，\(Q^* = 104\)

(b) \(E_d = (-4) \times \frac{24}{104} = -0.92\) （缺乏弹性）

(c) 税后供给 \(Q_s = -70 + 6P\)，新均衡 \(P^* = 27\)，\(Q^* = 92\)

题目 2：弧弹性计算与总收益判断

某商品价格从100元降至90元，销量从1000件增至1300件。

解答

(a) 中点法：\(E_d = \frac{300/1150}{-10/95} = -2.48\)

(b) \(|E_d| = 2.48 > 1\)，弹性需求

(c) TR: 100,000 → 117,000，收益增加17,000（弹性需求降价增收）

题目 3：交叉价格弹性计算

咖啡价格上涨10%导致茶的需求量增加15%。

解答

\(E = 15\% / 10\% = 1.5\)。正值确认为替代品，数值1.5表明替代程度较高。

分析题：2008年房价下跌为何成交量也下降？

关键洞见

不是"沿需求曲线移动" — 是需求曲线本身左移：

收入下降+失业 = 购买力减少；信贷收紧 = 贷款困难；预期继续跌 = 持币观望

同时供给也在调整：负资产卖家无法出售。

观察到的是两条变化后曲线的新均衡，而非沿原曲线的移动。
