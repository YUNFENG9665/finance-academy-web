核心概念

预测 vs 因果推断

两种范式的根本区别
目标
预测（Prediction）: 预测\(Y\)的值
因果推断（Causal Inference）: 估计\(X\)对\(Y\)的因果效应
关注
预测（Prediction）: \(\hat{Y}\)的准确性
因果推断（Causal Inference）: \(\hat{\beta}\)的无偏性
变量选择
预测（Prediction）: 提升预测力即可
因果推断（Causal Inference）: 需要因果图指导
相关性
预测（Prediction）: 有用
因果推断（Causal Inference）: 不充分
模型复杂度
预测（Prediction）: 可以很复杂
因果推断（Causal Inference）: 偏好简单解释
评估
预测（Prediction）: 样本外预测误差
因果推断（Causal Inference）: 识别策略可信度
偏差-方差权衡（Bias-Variance Tradeoff）
\[E[(Y - \hat{Y})^2] = \underbrace{Bias(\hat{Y})^2}_{\text{偏差}} + \underbrace{Var(\hat{Y})}_{\text{方差}} + \underbrace{\sigma^2}_{\text{不可约误差}}\]
•简单模型：高偏差、低方差
•复杂模型：低偏差、高方差
•机器学习：寻找最优权衡点

正则化方法

过拟合问题

模型在训练集表现好，但在新数据上表现差。

惩罚回归的一般形式
\[\min_{\beta} \sum_{i=1}^{n}(Y_i - X_i'\beta)^2 + \lambda \cdot Penalty(\beta)\]
LASSO（L1正则化）
\[\min_{\beta} \sum_{i=1}^{n}(Y_i - X_i'\beta)^2 + \lambda \sum_{j=1}^{p}|\beta_j|\]

特点：

•产生稀疏解（部分\(\beta_j = 0\)）
•自动变量选择
•适合高维数据
Ridge回归（L2正则化）
\[\min_{\beta} \sum_{i=1}^{n}(Y_i - X_i'\beta)^2 + \lambda \sum_{j=1}^{p}\beta_j^2\]

特点：

•系数收缩但不为零
•处理多重共线性
•数值稳定
弹性网（Elastic Net）
\[\min_{\beta} \sum_{i=1}^{n}(Y_i - X_i'\beta)^2 + \lambda_1 \sum|\beta_j| + \lambda_2 \sum\beta_j^2\]

结合L1和L2的优点。

调参方法
交叉验证
描述: 最常用，选择使CV误差最小的\(\lambda\)
信息准则
描述: AIC、BIC选择
1SE规则
描述: 选择CV误差在1标准误内的最简模型

树模型

决策树（Decision Tree）

递归二分：

\[\min_{j,s} \left[\min_{c_1} \sum_{x_i \in R_1} (y_i - c_1)^2 + \min_{c_2} \sum_{x_i \in R_2} (y_i - c_2)^2\right]\]

其中\(R_1 = \{X|X_j \leq s\}\)，\(R_2 = \{X|X_j > s\}\)

剪枝（Pruning）
\[\min_T \sum_{m=1}^{|T|}\sum_{i \in R_m}(y_i - \hat{y}_{R_m})^2 + \alpha|T|\]

\(\alpha\)控制复杂度惩罚。

随机森林（Random Forest）
1.Bootstrap采样\(B\)个训练集
2.每个训练集建一棵树，每次分裂随机选\(m\)个特征
3.预测：回归取平均，分类取投票
\[\hat{f}_{RF}(x) = \frac{1}{B}\sum_{b=1}^{B}T_b(x)\]
优点
•降低方差（集成效应）
•不易过拟合
•可计算变量重要性
梯度提升树（GBDT/XGBoost）

顺序训练，每棵树拟合前面树的残差：

\[F_m(x) = F_{m-1}(x) + \nu \cdot h_m(x)\]

其中\(h_m(x)\)拟合\(-\frac{\partial L(y, F_{m-1}(x))}{\partial F_{m-1}(x)}\)

XGBoost目标函数
\[Obj = \sum_{i=1}^{n}L(y_i, \hat{y}_i) + \sum_{k=1}^{K}\Omega(f_k)\]
\[\Omega(f) = \gamma T + \frac{1}{2}\lambda\sum_{j=1}^{T}w_j^2\]

神经网络

多层感知机（MLP）
\[\hat{Y} = f^{(L)}(W^{(L)} \cdot f^{(L-1)}(W^{(L-1)} \cdot ... f^{(1)}(W^{(1)} X + b^{(1)})...) + b^{(L)})\]
激活函数
Sigmoid
公式: \(\sigma(x) = \frac{1}{1+e^{-x}}\)
特点: 输出(0,1)，梯度消失
Tanh
公式: \(\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}\)
特点: 输出(-1,1)
ReLU
公式: \(\max(0, x)\)
特点: 最常用，计算简单
Softmax
公式: \(\frac{e^{x_i}}{\sum_j e^{x_j}}\)
特点: 多分类输出层
反向传播（Backpropagation）

使用链式法则计算梯度，梯度下降更新参数。

深度学习在金融中的应用

机器学习因果推断

双重机器学习（Double/Debiased ML）

解决高维控制变量问题：

\[Y = D\theta + g_0(X) + U\]
\[D = m_0(X) + V\]
步骤
1.用ML估计\(\hat{g}_0(X)\)（\(Y\)对\(X\)的条件期望）
2.用ML估计\(\hat{m}_0(X)\)（\(D\)对\(X\)的条件期望）
3.用残差回归：\(Y - \hat{g}_0(X) = (D - \hat{m}_0(X))\theta + error\)

正交化消除正则化偏差。

因果森林（Causal Forest）

估计条件平均处理效应（CATE）：

\[\tau(x) = E[Y(1) - Y(0)|X=x]\]

修改随机森林的分裂准则，最大化处理效应的异质性。

应用场景
•个性化定价
•精准营销
•政策异质性效应

模型可解释性的重要性
•监管要求（如GDPR）

•模型验证
•业务洞察

•局部可加性
•一致性

关键算法与实现

1

LASSO路径算法

坐标下降法

对每个系数\(\beta_j\)：

\[\beta_j^{new} = \text{soft}(\beta_j^{LS}, \lambda)\]

其中软阈值函数：

\[\text{soft}(z, \lambda) = \text{sign}(z)(|z| - \lambda)_+\]
2

梯度提升伪代码

初始化 F_0(x) = 0
for m = 1 to M:
    计算残差: r_i = y_i - F_{m-1}(x_i)
    拟合树 h_m 到残差
    更新: F_m(x) = F_{m-1}(x) + ν * h_m(x)
return F_M(x)

Q2: 为什么LASSO能实现变量选择而Ridge不能？

A:

几何解释
•LASSO：L1约束区域是菱形，角点对应稀疏解
•Ridge：L2约束区域是圆形，解通常不在坐标轴上
数学原因

L1在原点不可微，存在次梯度条件导致精确的零解。

\[\hat{\beta}_j^{LASSO} = 0 \quad \text{当} \quad |X_j'(Y - X_{-j}\beta_{-j})| < \lambda\]
实践影响
•高维数据首选LASSO
•多重共线性时Ridge更稳定
•弹性网结合两者优点

Q6: 如何解释神经网络的预测结果？

挑战

神经网络是"黑箱"，难以直接解释。

解决方案

1. 特征重要性

•梯度基方法（Saliency Maps）

2. SHAP值

\[\phi_j(f, x) = \sum_{S} \frac{|S|!(p-|S|-1)!}{p!}[f(S \cup \{j\}) - f(S)]\]
•可加性：\(f(x) = \sum_j \phi_j + E[f(X)]\)
•局部一致性

3. LIME（局部可解释）

在预测点附近拟合线性模型：

\[\xi = \arg\min_{g \in G} L(f, g, \pi_x) + \Omega(g)\]

4. 注意力机制

Transformer架构提供的注意力权重可视化。

Q8: 双重机器学习（Double ML）解决了什么问题？

A:

问题背景

高维控制变量下估计处理效应。

传统方法的问题
•OLS：维度灾难
•LASSO直接用：正则化偏差
Double ML的解决方案
1.正交化

消除估计\(g_0(X)\)和\(m_0(X)\)的偏差对\(\theta\)的影响

2.Neyman正交矩
\[\psi(W; \theta, \eta) = (Y - g_0(X) - (D - m_0(X))\theta)(D - m_0(X))\]
