核心概念

平稳性

弱平稳（协方差平稳）的条件
均值恒定
数学表达: \(E[Y_t] = \mu, \forall t\)
含义: 期望不随时间变化
方差恒定
数学表达: \(Var(Y_t) = \sigma^2, \forall t\)
含义: 波动不随时间变化
自协方差只取决于时滞
数学表达: \(Cov(Y_t, Y_{t-k}) = \gamma_k\)
含义: 动态关系时不变

强平稳：\((Y_t, Y_{t+1}, ..., Y_{t+k})\)的联合分布与\(t\)无关。

自相关函数（ACF）
\[\rho_k = \frac{\gamma_k}{\gamma_0} = \frac{Cov(Y_t, Y_{t-k})}{Var(Y_t)}\]
偏自相关函数（PACF）

控制中间变量后的相关性，是\(Y_t\)对\(Y_{t-k}\)回归中\(Y_{t-k}\)的系数。

单位根检验

单位根过程
\[Y_t = \phi Y_{t-1} + \epsilon_t\]

当\(\phi = 1\)时为随机游走，非平稳。

ADF检验（Augmented Dickey-Fuller）

回归方程：

\[\Delta Y_t = \alpha + \beta t + \gamma Y_{t-1} + \sum_{i=1}^{p}\delta_i \Delta Y_{t-i} + \epsilon_t\]
•\(H_0: \gamma = 0\)（存在单位根，非平稳）
•\(H_1: \gamma < 0\)（平稳）
检验统计量
\[ADF = \frac{\hat{\gamma}}{se(\hat{\gamma})}\]

使用Dickey-Fuller分布临界值，非标准正态。

PP检验（Phillips-Perron）

对序列相关进行非参数调整，适用于异方差情况。

KPSS检验
•\(H_0\)：序列平稳
•\(H_1\)：存在单位根

与ADF互补使用，降低检验误差。

ARMA模型

AR(p)模型
\[Y_t = c + \phi_1 Y_{t-1} + \phi_2 Y_{t-2} + ... + \phi_p Y_{t-p} + \epsilon_t\]

平稳条件：特征方程\(1 - \phi_1 z - \phi_2 z^2 - ... - \phi_p z^p = 0\)的根都在单位圆外。

MA(q)模型
\[Y_t = \mu + \epsilon_t + \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + ... + \theta_q \epsilon_{t-q}\]

可逆条件：特征方程的根都在单位圆外。

ARMA(p,q)模型
\[Y_t = c + \sum_{i=1}^{p}\phi_i Y_{t-i} + \epsilon_t + \sum_{j=1}^{q}\theta_j \epsilon_{t-j}\]
滞后算子表示
\[\Phi(L)Y_t = c + \Theta(L)\epsilon_t\]
模型识别
AR(p)
ACF: 指数衰减
PACF: p阶后截尾
MA(q)
ACF: q阶后截尾
PACF: 指数衰减
ARMA(p,q)
ACF: 指数衰减
PACF: 指数衰减

模型选择与诊断

信息准则
\[AIC = -2\ln(L) + 2k\]
\[BIC = -2\ln(L) + k\ln(n)\]

其中\(L\)为似然函数值，\(k\)为参数数，\(n\)为样本量。

BIC对参数惩罚更重，倾向于选择更简洁的模型。

残差诊断

Ljung-Box检验（残差自相关）：

\[Q(m) = n(n+2)\sum_{k=1}^{m}\frac{\hat{\rho}_k^2}{n-k} \sim \chi^2_m\]
•\(H_0\)：残差无自相关
•拒绝\(H_0\)：模型设定不当
ARCH效应检验
\[\hat{\epsilon}_t^2 = \alpha_0 + \alpha_1 \hat{\epsilon}_{t-1}^2 + ... + \alpha_q \hat{\epsilon}_{t-q}^2 + u_t\]
\[LM = nR^2 \sim \chi^2_q\]

GARCH模型族

GARCH(1,1)模型

均值方程：

\[r_t = \mu + \epsilon_t\]

方差方程：

\[\sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2\]
参数约束
•\(\omega > 0\)
•\(\alpha \geq 0, \beta \geq 0\)
•\(\alpha + \beta < 1\)（方差平稳）
条件方差的解释
•\(\omega/(1-\alpha-\beta)\)：长期方差
•\(\alpha\)：短期冲击影响
•\(\beta\)：波动持续性
•\(\alpha + \beta\)：波动半衰期
GARCH扩展模型
EGARCH
方差方程: \(\ln\sigma_t^2 = \omega + \alpha g(z_{t-1}) + \beta\ln\sigma_{t-1}^2\)
特点: 杠杆效应
GJR-GARCH
方差方程: \(\sigma_t^2 = \omega + (\alpha + \gamma I_{t-1})\epsilon_{t-1}^2 + \beta\sigma_{t-1}^2\)
特点: 非对称效应
TGARCH
方差方程: 类似GJR
特点: 门限效应
IGARCH
方差方程: \(\alpha + \beta = 1\)
特点: 积分GARCH

EGARCH模型（Nelson 1991）：

\[\ln\sigma_t^2 = \omega + \alpha(|z_{t-1}| - E|z_{t-1}|) + \gamma z_{t-1} + \beta\ln\sigma_{t-1}^2\]

其中\(z_t = \epsilon_t/\sigma_t\)，\(\gamma < 0\)表示杠杆效应。

VAR模型

VAR(p)模型
\[\mathbf{Y}_t = \mathbf{c} + \mathbf{A}_1 \mathbf{Y}_{t-1} + \mathbf{A}_2 \mathbf{Y}_{t-2} + ... + \mathbf{A}_p \mathbf{Y}_{t-p} + \boldsymbol{\epsilon}_t\]

其中\(\mathbf{Y}_t\)为\(n \times 1\)向量，\(\mathbf{A}_i\)为\(n \times n\)系数矩阵。

平稳条件

\(|\mathbf{I} - \mathbf{A}_1 z - \mathbf{A}_2 z^2 - ... - \mathbf{A}_p z^p| = 0\)的根都在单位圆外。

脉冲响应函数（IRF）
\[\frac{\partial Y_{j,t+s}}{\partial \epsilon_{i,t}}\]

测量变量\(i\)的一个标准差冲击对变量\(j\)在\(s\)期后的影响。

正交化脉冲响应

使用Cholesky分解：\(\boldsymbol{\Sigma} = \mathbf{P}\mathbf{P}'\)

方差分解
\[Y_{j,t+s}MATH0i$冲击解释的比例。 **Granger因果检验**： 检验MATH1是否Granger因果于MATH2： \]Y_t = \alpha + \sum_{i=1}^{p}\beta_i Y_{t-i} + \sum_{i=1}^{p}\gamma_i X_{t-i} + \epsilon_t\[ - \(H_0: \gamma_1 = \gamma_2 = ... = \gamma_p = 0\) - 使用F检验

协整与误差修正模型

协整

两个I(1)序列的线性组合是I(0)。

\]Y_t \sim I(1), X_t \sim I(1)\[

\]Y_t - \beta X_t = u_t \sim I(0)\[

则称\(Y_t\)和\(X_t\)协整，协整向量为\((1, -\beta)\)。

Engle-Granger两步法
1.OLS估计长期关系：\(Y_t = \alpha + \beta X_t + u_t\)
2.检验残差\(\hat{u}_t\)的平稳性（ADF检验）
误差修正模型（ECM）

\]\Delta Y_t = \alpha + \gamma(Y_{t-1} - \beta X_{t-1}) + \sum_i \delta_i \Delta Y_{t-i} + \sum_j \phi_j \Delta X_{t-j} + \epsilon_t\[

•\(\gamma < 0\)：误差修正机制
•\(|\gamma|\)：调整速度
•\((Y_{t-1} - \beta X_{t-1})\)：上期偏离长期均衡的程度
Johansen协整检验

基于VAR的系统方法，可检验多变量协整关系。

关键公式与估计方法

1

ARMA模型的估计

条件最小二乘法（AR模型）：

\]\min_{\phi} \sum_{t=p+1}^{T}(Y_t - c - \phi_1 Y_{t-1} - ... - \phi_p Y_{t-p})^2\[

最大似然估计

假设\(\epsilon_t \sim N(0, \sigma^2)\)，最大化对数似然函数。

预测
一步预测

\]\hat{Y}_{T+1|T} = E[Y_{T+1}|Y_T, Y_{T-1}, ...]\[

多步预测

迭代使用预测值替代未来观测。

预测误差方差随预测期增加而增加。

2

GARCH模型的估计

拟似然估计（QMLE）

对数似然函数：

\]\ell(\theta) = -\frac{1}{2}\sum_{t=1}^{T}\left[\ln\sigma_t^2 + \frac{\epsilon_t^2}{\sigma_t^2}\right]\[

即使误差非正态，QMLE也是一致的。

波动率预测

\]\hat{\sigma}_{T+1|T}^2 = \omega + \alpha \epsilon_T^2 + \beta \sigma_T^2\[

\]\hat{\sigma}_{T+h|T}^2 = \frac{\omega}{1-\alpha-\beta} + (\alpha+\beta)^{h-1}(\sigma_{T+1|T}^2 - \frac{\omega}{1-\alpha-\beta})\[

长期预测收敛到无条件方差。

3

模型选择流程

1. 数据可视化
   └─ 时序图、ACF图、PACF图

2. 平稳性检验
   ├─ ADF/PP/KPSS检验
   └─ 必要时差分

3. 模型识别
   ├─ ACF/PACF确定p, q
   └─ 信息准则比较

4. 参数估计
   └─ MLE/条件MLE

5. 诊断检验
   ├─ 残差白噪声检验
   └─ ARCH效应检验

6. 波动率建模（如需要）
   └─ GARCH族模型

7. 预测与评估
   └─ 样本外预测评估

关键问题解答

概念类问题

Q1: 为什么平稳性在时间序列分析中如此重要？

A:

平稳性的重要性
1.统计推断有效性
•非平稳序列的均值、方差不恒定
•传统统计量（如t统计量）失效
•可能得到"伪回归"
2.预测的可行性
•平稳过程的统计特性稳定
•可以用历史数据预测未来
•非平稳过程预测困难
3.模型解释的意义
•AR系数有稳定的经济含义
•脉冲响应有限收敛
处理非平稳的方法
•差分使序列平稳
•协整处理多变量非平稳
•使用收益率而非价格

Q2: GARCH模型为什么能捕捉波动聚集？

A:

波动聚集的本质

大波动后倾向于跟随大波动，数学上表现为\(\epsilon_t^2\)序列正自相关。

GARCH的机制

\]\sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2\[

1.当\(\epsilon_{t-1}^2\)大时，\(\sigma_t^2\)增加
2.由于\(\beta\)通常较大，高波动会持续
3.形成波动的自我强化过程
参数的角色
•\(\alpha\)：新信息的影响权重
•\(\beta\)：历史波动的持续性
•\(\alpha + \beta\)接近1：强波动聚集
验证

GARCH残差\(z_t = \epsilon_t/\sigma_t\)的平方应该无自相关。

Q3: 什么是协整？为什么不能直接对I(1)序列做回归？

A:

直接回归I(1)的问题
1.伪回归（Spurious Regression）
•两个独立的随机游走回归可能显著
•t统计量不收敛到标准分布
•R²可能很高但无意义
2.统计量失效
•OLS估计量是一致的（超级一致）
•但标准误公式错误
•推断完全不可靠
协整的意义

如果\(Y_t\)和\(X_t\)协整：

•它们有共同的随机趋势
•存在长期稳定的均衡关系
•可以正确估计和推断
判断方法
1.先检验各序列的阶数
2.如果都是I(1)，检验残差是否I(0)
3.存在协整则使用ECM
应用类问题

Q4: 如何选择ARMA模型的阶数p和q？

A:

方法一：ACF/PACF判断

模式ACFPACF模型
截尾✓-MA
-截尾✓AR
拖尾拖尾拖尾ARMA

方法二：信息准则

1.估计多个备选模型
2.比较AIC/BIC
3.选择信息准则最小的

方法三：一般到特殊

1.从较高阶开始
2.逐步删除不显著的滞后项
3.直到所有系数显著
实践建议
•金融数据通常低阶即可（p, q ≤ 2）
•信息准则和诊断检验结合使用
•样本外预测能力最重要

Q5: VAR模型中的因果关系能否解释为真正的因果？

A:

Granger因果的局限
1.只是预测性因果
•X Granger因果Y：X的滞后有助于预测Y
•不意味X导致Y
2.可能存在混淆
•第三变量Z同时影响X和Y
•Z领先于X和Y
3.信息集不完整
•遗漏重要变量可能改变结论
•变量选择影响结果
正确理解
•Granger因果是统计概念
•表示信息流动方向
•真正因果需要经济理论和实验证据

Q6: 如何用GARCH模型计算VaR？

A:

VaR定义

\]P(R_t < VaR_\alpha) = \alpha\[

GARCH-VaR计算
1.估计GARCH模型

获得条件波动率预测\(\hat{\sigma}_{t+1|t}\)

2.假设分布
•正态分布：\(VaR_\alpha = \mu - z_\alpha \hat{\sigma}_{t+1|t}\)
•t分布：使用t分布分位数
•经验分布：使用历史分位数
3.滚动预测

每天更新波动率预测和VaR

回测检验
•统计VaR突破次数
•应该接近理论频率\(\alpha\)
•Kupiec检验、Christoffersen检验
示例

如果\(\hat{\sigma}_{t+1|t} = 1.5\%\)，\(\mu = 0.02\%\)

\]VaR_{0.05} = 0.02\% - 1.645 \times 1.5\% = -2.45\%$$

批判性问题

Q7: 金融时间序列是否可预测？有效市场假说如何解释？

A:

实证证据
收益率水平
可预测性: 很弱
经济显著性: 扣除成本后消失
波动率
可预测性: 强
经济显著性: GARCH有效
协整关系
可预测性: 存在
经济显著性: 套利有限
与EMH的协调
1.弱式有效
•不能用历史价格预测收益
•与ARMA对收益的弱预测力一致
2.可预测但不可套利
•交易成本
•市场冲击
•风险调整后无超额收益
3.风险溢价时变
•可预测性可能反映风险变化
•不违反EMH

Q8: 为什么很多GARCH模型估计出的\(\alpha+\beta\)接近1？

A:

观察现象

实证中\(\alpha + \beta\)通常在0.95-0.99。

可能解释
1.真实的高持续性
•金融波动确实高度持续
•半衰期可达数月
2.结构变化
•忽略均值和方差的结构变化
•导致高持续性估计
3.长记忆
•真实过程可能是分数积分
•GARCH误设为高持续性
4.杠杆效应
•未考虑非对称性
•导致持续性高估
应对方法
•使用FIGARCH（分数积分）
•加入结构断点
•使用非对称GARCH

Q9: 机器学习方法与传统时间序列方法如何比较？

平稳性检验|ADF·KPSS
▶平稳: 均值/方差不随时间变化
▶ADF检验: 零假设为存在单位根
▶非平稳→差分: 使序列变平稳
模型识别|ACF·PACF
▶AR(p): PACF在p阶后截尾
▶MA(q): ACF在q阶后截尾
▶ARMA(p,q): 两者都拖尾
参数估计|AIC·BIC
▶最大似然估计参数值
▶AIC/BIC: 模型复杂度vs拟合度权衡
▶诊断: Ljung-Box检验残差白噪声
预测应用|金融时序

▶VAR: 多变量时序联合预测
▶协整: 非平稳序列的长期均衡关系
