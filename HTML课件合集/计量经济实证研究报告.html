Winsorization处理
\[X_i^{win} = \begin{cases} P_1 & \text{if } X_i < P_1 \\ X_i & \text{if } P_1 \leq X_i \leq P_{99} \\ P_{99} & \text{if } X_i > P_{99} \end{cases}\]
4

变量定义与度量

变量分类
因变量(Y)
角色: 要解释的结果
选择原则: 准确度量研究对象
自变量(X)
角色: 核心解释变量
选择原则: 准确度量处理/暴露
控制变量
角色: 混淆因素
选择原则: 基于理论和因果图
工具变量
角色: 识别外生变异
选择原则: 相关性+外生性
常见金融变量的构造
Tobin's Q
定义: (股票市值+负债)/总资产
注意事项: 不同定义影响结果
ROA
定义: 净利润/总资产
注意事项: 使用期初/期末/平均资产
杠杆率
定义: 负债/资产
注意事项: 账面/市值杠杆区别
公司规模
定义: ln(总资产)或ln(市值)
注意事项: 取对数减少偏度
研发强度
定义: R&D/销售收入
注意事项: 处理缺失值
变量标准化
\[Z_i = \frac{X_i - \bar{X}}{s_X}\]

便于比较不同变量的系数大小。

5

计量模型设定

基准回归模型
\[Y_{it} = \beta_0 + \beta_1 X_{it} + \gamma' \mathbf{Controls}_{it} + \alpha_i + \delta_t + \epsilon_{it}\]
\(Y_{it}\)
含义: 因变量
选择: 根据研究问题
\(X_{it}\)
含义: 核心解释变量
选择: 关注\(\beta_1\)
\(\mathbf{Controls}\)
含义: 控制变量
选择: 理论指导
\(\alpha_i\)
含义: 个体固定效应
选择: 控制时不变异质性
\(\delta_t\)
含义: 时间固定效应
选择: 控制共同冲击
模型选择决策树
数据类型？
├─ 截面数据
│  └─ OLS + 稳健标准误
├─ 面板数据
│  ├─ 关注个体异质性 → 固定效应
│  ├─ 估计时不变变量效应 → 随机效应
│  └─ Hausman检验指导选择
└─ 时间序列
   └─ 检验平稳性 → ARMA/VAR

稳健性检验

稳健性检验的目的

证明结论不依赖于特定的模型设定或样本选择。

常见稳健性检验
替代变量
方法: 换因变量/自变量定义
目的: 排除度量误差
替代样本
方法: 子样本、排除极端值
目的: 排除样本驱动
替代模型
方法: 不同固定效应、聚类
目的: 排除模型设定
安慰剂检验
方法: 假处理/假时间
目的: 验证识别有效
替代识别
方法: 不同工具变量
目的: 验证因果识别

机制分析

机制分析的目的

打开"黑箱"，解释X如何影响Y。

机制检验方法

1. 中介效应检验

\[Y = c X + e_1 \quad \text{(总效应)}\]
\[M = a X + e_2 \quad \text{(X对中介的效应)}\]
\[Y = c' X + b M + e_3 \quad \text{(直接效应)}\]

中介效应 = \(a \times b\) 或 \(c - c'\)

2. 异质性分析

\[Y = \beta_1 X + \beta_2 X \times Z + \gamma Z + controls + \epsilon\]

\(\beta_2\)显著：X的效应因Z而异。

3. 分样本检验

按特征分组（如国企vs民企），检验效应差异。

案例一：公司治理与企业绩效的实证研究

研究问题

独立董事比例是否影响公司绩效？

理论与假说
•代理理论：独立董事监督管理层，提升绩效
•资源依赖理论：独立董事提供外部资源
•H1：独立董事比例与公司绩效正相关
数据与样本
•来源：CSMAR数据库
•期间：2010-2023年
•样本：A股非金融上市公司
•观测：约30,000公司-年

案例二：并购公告的事件研究

研究问题

并购公告对收购方股价的短期影响如何？

数据
•并购事件：A股2015-2023年重大资产重组
•样本：1,234个并购公告
•股票数据：日度收益率

方法：事件研究法

第一步：估计正常收益

估计窗口：[-250, -21]

\[R_{it} = \alpha_i + \beta_i R_{mt} + \epsilon_{it}\]

第二步：计算异常收益

\[AR_{it} = R_{it} - (\hat{\alpha}_i + \hat{\beta}_i R_{mt})\]

第三步：累计异常收益

\[CAR_i[t_1, t_2] = \sum_{t=t_1}^{t_2} AR_{it}\]

第四步：统计检验

\[t_{CAR} = \frac{\overline{CAR}}{\sigma_{CAR}/\sqrt{N}}\]
结果

\[CAR_i = \beta_0 + \beta_1 RelativeSize_i + \beta_2 Cash_i + \beta_3 Related_i + \epsilon_i\]
相对规模
系数: -0.034***

2.交易规模越大，负效应越明显
3.现金支付和同行业并购表现较好

关键问题解答

概念类问题

Q1: 什么是好的识别策略？如何评价一篇实证论文的因果识别？

A:

好的识别策略特征
清晰的识别假设
描述: 明确说明依赖的假设
可检验的前提
描述: 部分假设可以验证
有说服力的论证
描述: 经济学直觉支持
稳健的结果
描述: 不同方法一致
评价清单
1.✓ 核心假设是什么？是否合理？
2.✓ 平行趋势/排除性约束是否满足？
3.✓ 是否有安慰剂检验？
4.✓ 稳健性检验是否全面？
5.✓ 外部效度如何？
常见问题
•没有明确识别假设
•稳健性检验流于形式
•对假设违反的讨论不足

Q2: 控制变量应该如何选择？越多越好吗？

A:

控制变量选择原则
混淆因素（同时影响X和Y）
不应该控制: 中介变量（在因果路径上）
前定变量（在X之前决定）
不应该控制: 结果变量（被Y影响）
不应该控制: "坏控制"（被处理影响）
过度控制的问题
1.控制中介变量低估真实效应
2.控制后定变量引入内生性
3.过多变量降低估计效率
实践建议
•基于因果图（DAG）选择
•报告不同控制变量集的结果
•讨论为什么包含/排除某些变量

Q3: 如何处理缺失数据？

A:

缺失机制
MCAR
定义: 完全随机缺失
处理: 删除或简单插值
MAR
定义: 随机缺失（条件于观测变量）
处理: 多重插补
MNAR
定义: 非随机缺失
处理: 需要建模选择机制
常用处理方法
1.列表删除：删除含缺失值的观测
2.均值/中位数插值：简单但可能引入偏差
3.多重插补：生成多个填补数据集
4.建模：将缺失作为信息利用
报告要求
•报告缺失情况
•解释处理方法
•检验结果对缺失处理的敏感性
应用类问题

Q5: 回归结果不显著怎么办？

A:

首先检查
1.样本量是否足够？
2.变量度量是否准确？
3.模型设定是否正确？
4.标准误是否适当？
不显著可能的原因
真的没有效应
对策: 这也是有价值的发现
样本量不足
对策: 扩大样本或用更精确度量
度量误差
对策: 改进变量定义
异质性抵消
对策: 分样本检验
模型设定不当
对策: 尝试非线性、交互项
报告不显著结果
•诚实报告，不要p-hacking
•讨论可能的原因
•不显著也是对文献的贡献

Q8: 如何看待"讲故事"与实证严谨性的关系？

A:

两者的张力
•好的论文需要compelling story
•但不能为了故事牺牲严谨性
•"讲故事"有时成为p-hacking的借口
正确的做法
理论先行，实证检验
不好的实践: 结果驱动，事后找理论
报告所有相关结果
不好的实践: 只报告支持故事的结果
讨论与预期不符的发现
不好的实践: 忽略不一致的证据
承认局限性
不好的实践: 过度解释结果
平衡策略
1.在研究开始时就明确假说
2.区分确认性分析和探索性分析
3.对于意外发现，承认是探索性的
4.让数据说话，不要强迫数据
