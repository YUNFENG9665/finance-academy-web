核心概念

1

有效市场假说（EMH）

理论框架

有效市场假说由Eugene Fama于1970年系统阐述，核心观点是在有效市场中，价格完全反映所有可用信息。

三种形式
形式信息集含义投资策略影响
弱式有效历史价格和成交量技术分析无效基本面分析可能有效
半强式有效所有公开信息基本面分析无效仅内幕信息可获利
强式有效所有信息（含内幕）任何分析都无效被动投资最优
EMH的理论基础
1.理性投资者：投资者能正确评估证券价值
2.随机交易：非理性行为相互抵消
3.套利力量：理性套利者消除错误定价
联合假设问题
•检验EMH必须同时假设一个"正确的"资产定价模型
•发现异常收益可能是市场无效，也可能是定价模型错误
2

资本资产定价模型（CAPM）

理论假设
投资者假设
具体内容: 风险规避、单期决策、均值-方差偏好
市场假设
具体内容: 完美市场、无摩擦、所有资产可分割
信息假设
具体内容: 同质预期、信息对称
其他假设
具体内容: 存在无风险资产、可无限制借贷
核心模型
\[E[R_i] = R_f + \beta_i(E[R_m] - R_f)\]

其中：

•\(E[R_i]\)：资产i的期望收益
•\(R_f\)：无风险收益率
•\(E[R_m]\)：市场组合期望收益
•\(\beta_i\)：资产i的系统性风险系数
贝塔的计算与解释
\[\beta_i = \frac{Cov(R_i, R_m)}{Var(R_m)} = \rho_{i,m} \times \frac{\sigma_i}{\sigma_m}\]
贝塔值风险特征典型行业投资含义
β > 1.5高度进攻型科技、券商牛市超额收益
1 < β < 1.5进攻型消费品、金融适度敏感
0.5 < β < 1防御型公用事业、医药熊市相对抗跌
β < 0.5高度防御型黄金、国债避险资产
3

市场异象与CAPM的挑战

主要市场异象
异象名称发现时间描述与CAPM矛盾可能解释
规模效应Banz(1981)小市值股票获得更高收益低β却有高收益流动性风险、信息不对称
价值效应Fama-French(1992)高B/M股票收益高非系统性风险溢价财务困境风险
动量效应Jegadeesh-Titman(1993)过去赢家继续赢3-12个月短期收益可预测投资者反应不足
反转效应De Bondt-Thaler(1985)长期输家变赢家（3-5年）长期可预测过度反应修正
低波动异象Ang et al.(2006)低β股票收益不低于高β直接违背CAPM杠杆约束、彩票偏好
盈利异象Novy-Marx(2013)高盈利公司收益更高不在传统风险框架内持续性误定价
投资异象Cooper et al.(2008)高投资公司收益低与公司特征相关过度投资、管理层自大
异象的三种解释范式
1.风险补偿派：异象反映的是被遗漏的风险因子
2.行为金融派：异象源于投资者的系统性认知偏差
3.数据挖掘派：异象是统计上的偶然发现

Fama-French多因子模型

三因子模型
\[R_i - R_f = \alpha_i + \beta_i^{MKT}(R_m - R_f) + \beta_i^{SMB} \cdot SMB + \beta_i^{HML} \cdot HML + \epsilon_i\]
因子构建方法
因子全称构建方式经济含义
MKTMarket\(R_m - R_f\)市场风险溢价
SMBSmall Minus Big小市值组合-大市值组合规模风险溢价
HMLHigh Minus Low高B/M组合-低B/M组合价值风险溢价
五因子模型扩展
\[R_i - R_f = \alpha + \beta_{MKT}MKT + \beta_{SMB}SMB + \beta_{HML}HML + \beta_{RMW}RMW + \beta_{CMA}CMA + \epsilon\]
新增因子全称含义理论基础
RMWRobust Minus Weak高盈利-低盈利盈利能力溢价
CMAConservative Minus Aggressive保守投资-激进投资投资谨慎溢价

其他多因子模型

Carhart四因子模型
\[R_i - R_f = \alpha + \beta_{MKT}MKT + \beta_{SMB}SMB + \beta_{HML}HML + \beta_{MOM}MOM + \epsilon\]
•MOM（Momentum）：过去12个月收益高的组合减去收益低的组合
AQR六因子模型

在五因子基础上增加质量因子（QMJ）和动量因子。

特征模型与因子模型的对比
核心逻辑

代表学者

特征模型: Daniel-Titman
经济解释
因子模型: 风险补偿
特征模型: 错误定价
实证表现

特征模型: 特征直接定价

因子投资与Smart Beta

因子投资的理论基础
可解释性
说明: 因子有清晰的经济逻辑
持久性
说明: 因子溢价长期存在
普遍性
说明: 在不同市场和时期有效
稳健性
说明: 在不同定义下仍然有效
可投资性
说明: 考虑交易成本后仍有超额收益
主要Smart Beta策略
策略类型目标因子构建方法历史年化超额
价值策略HML买入低估值股票3-5%
动量策略MOM买入近期涨幅大的股票5-8%
低波动策略Low Vol买入低波动率股票2-4%
质量策略QMJ买入高盈利、低杠杆公司3-5%
规模策略SMB倾斜小盘股2-3%
因子拥挤与容量约束

当过多资金追逐同一因子时，会出现：

•因子估值过高
•预期超额收益下降
•因子崩溃风险增加

关键模型与公式

1

模型1：证券市场线（SML）

\[E[R_i] = R_f + \beta_i \times (E[R_m] - R_f)\]
•所有资产都应位于SML上
•阿尔法>0的资产被低估，应该买入
•阿尔法<0的资产被高估，应该卖出
2

模型2：夏普比率与信息比率

\[SR = \frac{E[R_p] - R_f}{\sigma_p}\]
\[IR = \frac{\alpha}{\sigma(\epsilon)} = \frac{E[R_p - R_b]}{\sigma(R_p - R_b)}\]
指标含义优秀水平应用场景
夏普比率单位总风险的超额收益>1绝对收益评估
信息比率单位主动风险的超额收益>0.5相对收益评估
3

模型3：因子模型回归

\[R_{it} - R_{ft} = \alpha_i + \sum_{k=1}^{K} \beta_{ik} F_{kt} + \epsilon_{it}\]
回归结果解读
•\(\alpha_i\)：无法被因子解释的异常收益
•\(\beta_{ik}\)：对因子k的敏感度
•\(R^2\)：因子模型的解释力度
4

模型4：因子协方差矩阵

\[Cov(R_i, R_j) = \sum_{k=1}^{K} \beta_{ik}\beta_{jk}\sigma_k^2 + Cov(\epsilon_i, \epsilon_j)\]

因子模型大幅简化了协方差矩阵的估计：从\(\frac{n(n-1)}{2}\)个参数减少到\(nK + K\)个参数。

5

模型5：因子择时模型

\[R_{it} - R_{ft} = \alpha_i + \beta_{i,t}F_t + \epsilon_{it}\]

其中\(\beta_{i,t}\)随时间变化：

\[\beta_{i,t} = \beta_i^0 + \beta_i^1 \cdot Z_{t-1}\]

\(Z_{t-1}\)为择时信号（如估值、宏观指标）。

背景

MSCI是全球最大的指数提供商之一，其因子指数被广泛用作Smart Beta投资的基准。

案例二：量化基金的因子归因分析

背景

某量化多空基金声称通过"独特的选股算法"获得超额收益，投资者希望了解其真实收益来源。

因子回归结果
R_fund - R_f = 0.2% + 0.05×MKT + 0.8×SMB - 0.3×HML + 0.6×MOM + 0.4×QMJ
                (t=0.8)  (t=0.5)   (t=6.2)   (t=-2.1)   (t=4.8)    (t=3.2)
R² = 0.72
归因结论
规模因子（SMB）
贡献: 月均1.6%
显著性: 高度显著
动量因子（MOM）
贡献: 月均0.9%
显著性: 高度显著
质量因子（QMJ）
贡献: 月均0.5%
显著性: 显著
价值因子（HML）
贡献: 月均-0.3%
显著性: 显著
阿尔法（α）
贡献: 月均0.2%
显著性: 不显著
关键结论
1.基金收益72%可被标准因子解释
2.真实阿尔法仅0.2%且不显著
3.主要收益来自规模和动量因子暴露
4."独特算法"实际是因子投资的包装

案例三：价值因子的十年低迷与争论

背景

2010-2020年价值因子（HML）经历了史上最长的低迷期，累计跑输成长股超过100%，引发"价值投资已死"的争论。

价值因子十年表现（2010-2020）
时期HML累计收益最大回撤相对成长股
2010-2015-8%-15%跑输32%
2015-2020-35%-45%跑输68%
整体-43%-52%跑输100%+
不同解释视角
价值已死派
核心观点: 低利率环境永久改变估值逻辑
证据支持: 长期债券收益率下降与成长股估值上升同步
周期论派
核心观点: 价值因子周期性低迷，终将回归
证据支持: 历史上曾有多次长期低迷后反弹
定义过时派
核心观点: 传统B/M指标无法捕捉现代经济的价值
证据支持: 科技公司无形资产被低估
拥挤消退派
核心观点: 因子被发现后超额收益消失
证据支持: 学术发表后因子表现普遍下降
2020-2022年的反转

疫情后通胀上升、利率反转，价值因子在2021-2022年大幅跑赢，支持了周期论的观点。

投资启示
1.任何因子都可能经历长期低迷
2.因子投资需要极强的纪律性和长期视野
3.理解因子背后的经济逻辑比追逐历史收益更重要

深度思考问题

概念理解类

Q1：为什么CAPM中只有系统性风险（β）得到补偿，而非系统性风险不被定价？

参考答案

在CAPM框架下，投资者可以通过分散化消除非系统性风险（个股特有风险）。由于这种风险可以"免费"消除：

1.理性投资者不会为可消除的风险承担溢价
2.均衡时，只有不可分散的系统性风险才需要补偿
3.系统性风险体现为与市场组合的协方差

因此，两只股票的预期收益差异完全由β决定，与总风险（σ）无关。

Q2：联合假设问题如何影响对EMH的检验？

参考答案

联合假设问题指的是：检验市场效率时，必须同时假设一个"正确的"资产定价模型。当发现异常收益时，存在两种可能：

1.市场是无效的，存在套利机会
2.市场是有效的，但使用的定价模型有误

这导致EMH在严格意义上是不可证伪的。例如，价值效应既可以解释为市场无效（投资者系统性低估价值股），也可以解释为价值股承担了某种未被CAPM捕捉的风险。

Q3：Fama-French三因子和五因子模型的经济解释是什么？

参考答案
三因子模型解释
•SMB：小公司面临更高的经营和流动性风险
•HML：高B/M公司多处于财务困境，有更高的违约风险
五因子模型解释
•RMW：高盈利公司被低估是因为投资者对盈利持续性不确定
•CMA：保守投资的公司更稳健，激进投资可能是过度自信

然而，这些解释仍有争议，行为金融学派认为这些异象源于投资者的系统性认知偏差。

应用分析类

Q4：如何构建一个多因子组合？因子权重如何确定？

参考答案
构建步骤
1.因子选择：选取有理论基础、实证稳健的因子（如MKT、SMB、HML、MOM、QMJ）
2.信号计算：对每只股票计算各因子暴露得分
3.组合构建：基于综合得分排序构建多空或纯多头组合
权重确定方法
等权法
说明: 各因子权重相等
优缺点: 简单，但忽略因子差异
波动率倒数
说明: 按因子波动率反比
优缺点: 风险均衡，低波动因子权重大
风险平价
说明: 各因子风险贡献相等
优缺点: 理论优美，实际需估计协方差
均值方差优化
说明: 基于预期收益和风险
优缺点: 对输入敏感，易过拟合
机器学习
说明: 数据驱动的动态权重
优缺点: 复杂，可能过拟合

Q5：如何使用因子模型进行业绩归因？

参考答案
回归方法
\[R_p - R_f = \alpha + \sum \beta_k F_k + \epsilon\]
归因步骤
1.选择合适的因子模型（如FF5）
2.运行时间序列回归
3.计算各因子的收益贡献：\(\beta_k \times \bar{F_k}\)
4.剩余的α为真实超额收益
解读要点
•因子暴露是免费的（可以通过被动投资获得）
•真正的技能体现在显著的正α
•R²反映模型解释力，过低说明有遗漏因子

批判思维类

Q7：如果市场异象是真实的定价错误，为什么不被套利消除？

参考答案

套利限制（Limits to Arbitrage）解释了为什么错误定价可以持续存在：

1.基本面风险：套利者可能在价格回归前被迫平仓
2.噪音交易者风险：错误定价可能在短期内恶化
3.实施成本：交易成本、借券费用、融资约束
4.模型风险：难以确定"正确"价格
5.代理问题：基金经理有业绩压力，难以承受短期亏损

经典案例：1998年LTCM危机中，尽管套利机会真实存在，但基金因杠杆和流动性问题在价格回归前破产。

Q8：因子溢价在学术发表后是否会消失？这对投资者意味着什么？

参考答案
实证证据
•McLean and Pontiff (2016) 研究发现：学术发表后，因子溢价平均下降约50%
•但大多数因子在发表后仍保留显著正收益
两种观点
会消失
论据: 套利资金涌入消除错误定价；因子拥挤推高估值
不会消失
论据: 风险补偿不会消失；行为偏差有生物学基础
对投资者的启示
1.不要期望获得历史回测中的全部超额收益
2.考虑实际可投资性（交易成本、容量）
3.关注因子的经济逻辑而非仅关注历史表现
4.分散于多个因子，避免集中押注

Q9：CAPM在实践中几乎不成立，为什么仍被广泛使用？

参考答案

尽管实证表现不佳，CAPM仍是金融实践的基石：

理由
1.概念简洁：提供了风险与收益关系的直觉框架
2.基准作用：作为比较其他模型的起点
3.监管认可：许多法规和会计准则基于CAPM
4.资本成本估算：尽管有偏，在实践中仍可用
5.教育价值：帮助理解金融学核心思想
使用建议
•认识到CAPM的局限性
•在重要决策中使用多种方法交叉验证
•将β视为风险的一个维度而非唯一维度
