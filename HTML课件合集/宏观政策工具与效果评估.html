丁伯根法则

N个独立目标需要N个独立工具。

•2个目标（通胀、就业）→ 需要2个工具（货币、财政）
•实际中工具可能不独立、目标可能冲突

政策传导机制

货币政策传导渠道
政策利率变化
    │
    ├──→ 利率渠道 ──→ 借贷成本 ──→ 投资/消费 ──→ 总需求
    │
    ├──→ 信贷渠道 ──→ 银行贷款供给 ──→ 企业投资 ──→ 总需求
    │
    ├──→ 资产价格渠道 ──→ 股价/房价 ──→ 财富效应/Q效应 ──→ 消费/投资
    │
    ├──→ 汇率渠道 ──→ 汇率变动 ──→ 净出口 ──→ 总需求
    │
    └──→ 预期渠道 ──→ 通胀预期 ──→ 实际利率/工资谈判 ──→ 消费/成本

各渠道重要性（美国估计）：

占比: ~25%
时滞: 6-18月
资产价格
占比: ~20%
时滞: 3-6月
汇率渠道
占比: ~10%
时滞: 3-9月
预期渠道
占比: ~5%
时滞: 即时
财政政策传导
政府支出↑ ──→ 直接增加总需求 ──→ 产出↑ ──→ 收入↑ ──→ 消费↑ ──→ 乘数效应
                                      │
                                      └──→ 利率↑ ──→ 投资↓ ──→ 挤出效应

减税 ──→ 可支配收入↑ ──→ 消费↑（取决于MPC）──→ 总需求↑
              │
              └──→ 储蓄↑（取决于永久收入判断）──→ 效果减弱

关键模型与公式

1

模型1：IS-LM模型

IS曲线（商品市场均衡）：

\[Y = C(Y-T) + I(r) + G\]

斜率：\(\frac{dr}{dY}|_{IS} = \frac{1-c}{I_r} < 0\)

LM曲线（货币市场均衡）：

\[\frac{M}{P} = L(Y, r)\]

斜率：\(\frac{dr}{dY}|_{LM} = \frac{L_Y}{L_r} > 0\)

政策效果
政策IS移动LM移动Y变化r变化
扩张性财政右移不变↑↑
扩张性货币不变右移↑↓
组合政策右移右移↑↑不确定
2

模型2：AD-AS模型

总需求(AD)
\[Y^d = f(P, M, G, T, ...)\]

AD曲线向下倾斜原因：

•实际余额效应（庇古效应）
•利率效应（凯恩斯效应）
•汇率效应（蒙代尔-弗莱明效应）
总供给(AS)
短期
形状: 向上倾斜
原因: 价格/工资粘性
长期
形状: 垂直
原因: 自然产出水平
短期AS
\[Y = Y^* + \alpha(P - P^e)\]

政策效果（AD-AS）：

扩张性政策
短期: Y↑, P↑
长期: 只有P↑
供给侧改革
短期: AS右移
长期: Y↑, P↓
3

模型3：泰勒规则

\[i_t = r^* + \pi_t + 0.5(\pi_t - \pi^*) + 0.5(y_t - y^*)\]
参数含义
\(r^*\)
含义: 均衡实际利率
典型值: 2%
\(\pi^*\)
含义: 目标通胀
典型值: 2%
0.5
含义: 通胀反应系数
典型值: 可调整
0.5
含义: 产出反应系数
典型值: 可调整
泰勒原则
\[\frac{\partial i}{\partial \pi} = 1 + 0.5 = 1.5 > 1\]

对通胀反应系数>1，确保实际利率上升，抑制需求。

实际央行行为
•美联储：大致遵循（有偏离）
•ECB：更关注通胀
•中国：非完全价格型调控
4

模型4：财政可持续性

政府债务动态
\[\frac{d_t - d_{t-1}}{d_{t-1}} = \frac{r - g}{1+g} - \frac{pb_t}{d_{t-1}}\]
•\(d\)：债务/GDP
•\(r\)：利率
•\(g\)：GDP增长率
•\(pb\)：基本盈余/GDP
债务可持续条件
\[pb \geq (r-g)d\]

基本盈余足以覆盖利息支出（考虑增长）。

r < g时

债务可持续性改善（但不能无限借债）

债务上限争议
•传统观点：债务/GDP有上限（如90%？）
•现代观点：取决于r与g关系、央行支持

案例3：日本安倍经济学

三支箭
第一支箭
内容: 大胆的货币政策（QQE）
效果: 日元贬值、股市上涨
第二支箭
内容: 灵活的财政政策
效果: 短期刺激
第三支箭
内容: 增长战略（结构改革）
效果: 进展有限
成效评估
2%通胀目标
结果: 长期未达成
评价: 失败
经济增长
结果: 温和复苏
评价: 一般
股市
结果: 大幅上涨
评价: 成功
日元贬值
结果: 显著
评价: 成功

为什么效果有限？

1.结构性问题（人口老龄化）货币政策无法解决
2.通缩预期根深蒂固
3.第三支箭（结构改革）力度不足
4.消费税上调抵消刺激效果
教训
•货币政策不是万能药
•结构改革与需求管理需配合
•预期管理非常困难
•人口因素是长期约束

核心问题

概念理解类
1.财政政策和货币政策的传导机制有何不同？哪个更有效？取决于什么条件？
•思考方向：直接vs间接、时滞、挤出效应
2.什么是"挤出效应"？在什么条件下挤出效应最严重？如何避免？
•思考方向：利率弹性、央行配合、产出缺口
3.为什么说"零利率下限"是货币政策的难题？有哪些应对措施？各有什么效果？
•思考方向：流动性陷阱、QE、负利率、财政配合
分析应用类
4.如果经济同时面临高通胀和高失业（滞胀），应该如何选择政策？
•思考方向：供给冲击、菲利普斯曲线、政策优先级
5.在开放经济中，财政政策和货币政策的效果如何受汇率制度影响？
•思考方向：蒙代尔-弗莱明、三元悖论、资本流动
6.中国的货币政策框架与发达国家有什么不同？为什么要采用结构性工具？
•思考方向：双轨制、传导机制、结构性问题
批判思维类
7.政府债务有上限吗？"现代货币理论"的观点可信吗？什么情况下会失败？
•思考方向：r vs g、通胀约束、汇率
8.中央银行应该关注资产价格泡沫吗？如何权衡金融稳定与价格稳定？
•思考方向：Greenspan put、宏观审慎、二元目标
9."规则"vs"相机抉择"：货币政策应该遵循固定规则还是灵活应对？
•思考方向：时间不一致性、可信度、灵活性
