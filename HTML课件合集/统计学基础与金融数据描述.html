离散程度指标
方差
公式: \(\sigma^2 = \frac{1}{n}\sum_{i=1}^{n}(X_i - \bar{X})^2\)
解释: 平均偏离程度的平方
标准差
公式: \(\sigma = \sqrt{\sigma^2}\)
解释: 与原数据单位相同

公式: \(CV = \frac{\sigma}{\bar{X}}\)
解释: 消除量纲，便于比较

公式: \(R = X_{max} - X_{min}\)
解释: 最简单的离散度量
分布形态指标
\[\text{偏度(Skewness)} = \frac{E[(X-\mu)^3]}{\sigma^3}\]
\[\text{峰度(Kurtosis)} = \frac{E[(X-\mu)^4]}{\sigma^4}\]
•偏度 = 0：对称分布
•偏度 > 0：右偏（正偏），右尾较长
•偏度 < 0：左偏（负偏），左尾较长
•峰度 = 3：正态分布（超额峰度 = 0）
•峰度 > 3：尖峰厚尾（leptokurtic）
•峰度 < 3：平峰（platykurtic）

金融数据的典型程式化事实（Stylized Facts）
1.尖峰厚尾（Leptokurtosis）
•收益率分布的峰度远高于正态分布
•极端事件发生频率高于正态预测
2.波动聚集（Volatility Clustering）
•大波动后倾向于跟随大波动
•小波动后倾向于跟随小波动
•数学表现：收益率不相关但平方收益率自相关
3.杠杆效应（Leverage Effect）
•负收益对波动率的影响大于同等正收益
•下跌行情波动率更高
4.长记忆（Long Memory）
•波动率的自相关函数缓慢衰减
•存在跨越较长时期的依赖性
5.均值回复（Mean Reversion）
•波动率倾向于向长期均值回归
•VIX指数的典型特征

假设检验

基本框架
原假设 \(H_0\)
定义: 需要检验的假设（通常是"没有效应"）
备择假设 \(H_1\)
定义: 与原假设相对的假设
检验统计量
定义: 基于样本计算的统计量
拒绝域
定义: 导致拒绝\(H_0\)的统计量取值区域
显著性水平 \(\alpha\)
定义: 第一类错误的概率（通常5%或1%）
p值
定义: 在\(H_0\)为真时观察到当前结果或更极端结果的概率
常用检验
单样本t检验
\[t = \frac{\bar{X} - \mu_0}{s/\sqrt{n}} \sim t_{n-1}\]

双样本t检验（独立）：

\[t = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{s_1^2/n_1 + s_2^2/n_2}}\]
配对样本t检验
\[t = \frac{\bar{D}}{s_D/\sqrt{n}}\]
两类错误
错误类型定义概率金融后果
第一类错误拒绝了正确的\(H_0\)\(\alpha\)错误地认为有效（如无效因子被采用）
第二类错误未拒绝错误的\(H_0\)\(\beta\)错失真正有效的策略

统计功效：\(Power = 1 - \beta\)

相关分析

Pearson相关系数
\[\rho_{XY} = \frac{Cov(X,Y)}{\sigma_X \sigma_Y} = \frac{E[(X-\mu_X)(Y-\mu_Y)]}{\sigma_X \sigma_Y}\]

样本相关系数：

\[r_{xy} = \frac{\sum_{i=1}^{n}(X_i - \bar{X})(Y_i - \bar{Y})}{\sqrt{\sum_{i=1}^{n}(X_i - \bar{X})^2}\sqrt{\sum_{i=1}^{n}(Y_i - \bar{Y})^2}}\]
相关系数的性质
•\(-1 \leq \rho \leq 1\)
•\(\rho = 0\)：线性不相关（不意味独立）
•\(|\rho| = 1\)：完全线性相关
Spearman秩相关系数
\[r_s = 1 - \frac{6\sum d_i^2}{n(n^2-1)}\]

其中\(d_i\)为秩次之差，对异常值更稳健。

相关不等于因果
•虚假相关（Spurious Correlation）
•混淆变量（Confounding Variables）
•反向因果（Reverse Causality）

简单回归分析

简单线性回归模型
\[Y_i = \alpha + \beta X_i + \epsilon_i\]
OLS估计
\[\hat{\beta} = \frac{\sum_{i=1}^{n}(X_i - \bar{X})(Y_i - \bar{Y})}{\sum_{i=1}^{n}(X_i - \bar{X})^2} = \frac{Cov(X,Y)}{Var(X)}\]
\[\hat{\alpha} = \bar{Y} - \hat{\beta}\bar{X}\]
拟合优度
\[R^2 = \frac{SSR}{SST} = 1 - \frac{SSE}{SST}\]

其中：

•\(SST = \sum(Y_i - \bar{Y})^2\)（总平方和）
•\(SSR = \sum(\hat{Y}_i - \bar{Y})^2\)（回归平方和）
•\(SSE = \sum(Y_i - \hat{Y}_i)^2\)（残差平方和）

关键问题解答

概念类问题

Q1: 为什么金融研究更常使用对数收益率而非简单收益率？

A: 对数收益率有几个重要优势：

1.时间可加性：连续期间的对数收益率可以直接相加
\[r_{t,t+2} = r_{t,t+1} + r_{t+1,t+2}\]
2.取值范围：对数收益率取值在\((-\infty, +\infty)\)，更适合正态分布假设
3.统计特性：对数收益率更接近对称分布
4.连续复利：与连续时间金融模型一致

但简单收益率在计算组合收益时更直观（截面可加性）。

Q2: 如何理解p值与显著性水平的关系？

A:

•显著性水平\(\alpha\)：事先设定的犯第一类错误的最大可接受概率
•p值：在原假设为真的条件下，观察到当前或更极端结果的概率

决策规则：

•p值 < \(\alpha\)：拒绝原假设
•p值 ≥ \(\alpha\)：不拒绝原假设

注意：

•p值不是原假设为真的概率
•统计显著不等于经济显著
•多重检验问题需要调整

Q3: 相关系数为零是否意味着两个变量独立？

A: 不一定。Pearson相关系数只测量线性关系。

•相关系数为零 → 线性不相关
•独立 → 相关系数为零
•线性不相关 ⊄ 独立

例如：\(Y = X^2\)，当\(X\)对称分布于0时，\(\rho_{XY} = 0\)，但\(X\)和\(Y\)显然不独立。

Q4: 如何判断股票收益率数据是否存在异常值？

A: 综合运用多种方法：

1.可视化检查
•箱线图识别离群点
•Q-Q图检验分布
2.统计方法
•3σ原则（适用于近似正态）
•IQR方法（更稳健）
•MAD（中位数绝对偏差）
3.业务判断
•是否有公司事件（除权、停牌）
•是否有数据录入错误
•是否有极端市场事件
4.处理建议
•确认数据错误：更正或删除
•真实极端值：Winsorize或保留并说明

Q6: 如何检验金融数据的平稳性？

A: 平稳性检验是时间序列分析的前提：

1.图形检验
•时序图观察趋势
•ACF图观察自相关衰减
2.单位根检验
•ADF检验（最常用）
•PP检验
•KPSS检验（原假设为平稳）
3.实务建议
•价格序列通常非平稳
•收益率序列通常平稳
•波动率序列可能存在长记忆
批判性问题

Q7: 正态分布假设在金融中的局限性是什么？

A: 主要局限：

1.低估尾部风险
•正态分布预测±3σ外事件概率0.3%
•实际金融市场远超此频率
•2008年金融危机多个"25σ事件"
2.忽视波动聚集
•正态假设波动恒定
•实际波动率高度时变
3.模型风险
•VaR低估实际风险
•期权定价存在系统偏差
•资本充足率可能不足
4.替代方案
•t分布（厚尾）
•混合正态分布
•极值理论（EVT）
•GARCH条件正态

Q8: 大样本和小样本的统计推断有何不同？

A:

分布
小样本（n<30）: 使用t分布
大样本（n>30）: 可用正态近似
中心极限定理
小样本（n<30）: 可能不适用
大样本（n>30）: 适用
标准误
小样本（n<30）: 需要更精确估计
大样本（n>30）: 渐近有效
稳健性
小样本（n<30）: 对假设敏感
大样本（n>30）: 较稳健
检验功效
小样本（n<30）: 较低
大样本（n>30）: 较高

金融数据通常样本较大，但：

•事件研究可能样本小
•极端事件样本有限
•非平稳数据有效样本缩短

Q9: 如何避免虚假相关导致的错误结论？

A: 识别和避免虚假相关：

1.时间趋势问题
•共同趋势导致高相关
•解决：使用差分或去趋势数据
2.遗漏变量偏差
•第三变量同时影响X和Y
•解决：控制潜在混淆因素
3.数据挖掘
•大量变量测试后偶然相关
•解决：样本外验证、调整p值
4.聚合偏差
•不同层面相关性不同
•解决：明确分析层面
5.因果方向
•相关不意味因果
•解决：理论指导、因果推断方法
